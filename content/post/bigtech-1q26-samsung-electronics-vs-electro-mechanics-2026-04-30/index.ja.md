*本稿では、4月29〜30日に発表されたビッグテック4社（Amazon、Microsoft、Alphabet、Meta）の1Q26決算を、韓国AIサプライチェーンの中核2銘柄——**サムスン電子 (005930) とサムスン電機 (009150)**——と結びつけて読み解く。基本的な視点はシンプルだ。同じ設備投資の数字が、一方には直接的な受注として、もう一方にはすでに価格に織り込まれた材料として作用する、という非対称性である。*

---

## エグゼクティブサマリー

- **ビッグテックが示すのはAI需要の失速ではなく、設備投資の再加速だ。** AWS +28%、Azure +39〜40%、Google Cloud +63%——3つのハイパースケーラークラウドが*同時に*加速している。4社合計の2026年AI・クラウド設備投資の年率換算は約**$650B**に達する。
- **サムスン電子は直接的な受益者だ。** 1Q26においてDS部門の営業利益が₩53.7T（全社営業利益の約94%）を占める中、ビッグテックの設備投資はHBM / DDR5 / eSSD / サーバーDRAMの受注にほぼ一対一で変換される。サムスン電子は設備投資の**受け取り側**であり、負担側ではない。
- **サムスン電機には構造的な追い風があるが、短期的な株価上値は重い。** AIサーバー向けMLCCおよびFC-BGA需要は本物の加速を見せているが、市場はすでに同社の評価軸を「スマートフォンサイクル部品」から「AIサーバー部品」へと切り替えた。ここからの追加上昇には、1四半期の好決算だけでなく、**2〜3四半期連続の業績上方修正**が必要条件となる。
- **6〜12カ月の相対選好：サムスン電子 > サムスン電機。** サムスン電子の既存保有者はホールドを推奨。新規参入は押し目待ち。サムスン電機は部分的な利益確定が合理的であり、現水準での追いかけ買いは効率が悪い。

## 1. 結論から

| 銘柄 | 判断 | 核心的根拠 |
| --- | --- | --- |
| **サムスン電子 (005930)** | **買い / ホールド推奨** | ビッグテックの設備投資がDS売上に直結。HBM4の顧客拡大とサーバーメモリ価格は上乗せ要因。出尽くしリスクは低い。 |
| **サムスン電機 (009150)** | **ホールド / 押し目待ち** | AI MLCCとFC-BGA の追い風は本物だが、成長期待のかなりの部分がすでに株価に折り込まれている。出尽くしリスクは中〜高。 |

両社の違いを一文で表すなら：

> **サムスン電子はAI設備投資の一次受益者であり、サムスン電機は二次受益者だ。** 一次受益とは、需要が損益計算書に直接反映されることを意味する。二次受益とは、売上より先に「物語」が株価に先行することを意味する。

## 2. ビッグテック 1Q26 の読み解き

今決算シーズンのメッセージは「AI投資の縮小」ではない。正反対だ。

| ハイパースケーラー | 主要指標 | 韓国サプライチェーンへの含意 |
| --- | --- | --- |
| **Amazon** | AWS売上 $37.6B、**前年比+28%**。2026年AI投資目標**$200B維持**；1Q設備投資 $44.2B。 | AWSサーバー増設 → HBM / DDR5 / eSSD / サーバーDRAM需要。Trainium / ASICサーバー拡張 → FC-BGA、高密度MLCC需要。 |
| **Microsoft** | Azure **+39〜40%**、AIの年率換算売上 $37B、3Q設備投資 $31.9B。 | Azureの設備不足解消 = メモリスループットの増加。サーバー・ネットワーク・電源レールMLCC需要の拡大。 |
| **Alphabet** | Google Cloud売上 $20B、**前年比+63%**、バックログ約$460B；2026年設備投資 $180〜190B。 | TPU / AIサーバー拡張 → HBM / DRAM / NAND / SSD需要。TPUおよびサーバーボード / ネットワークFC-BGA + MLCC需要。 |
| **Meta** | 売上 $56.3B（+33%）、広告インプレッション+19%、広告単価+12%。2026年設備投資ガイダンスを**$125〜145Bへ引き上げ**。 | AIデータセンターの継続的な増設がメモリ需要を押し上げ。部品価格のレバレッジ環境は良好。 |

注目すべき観察が3点ある。

**観察1 — 3クラウドが*同時に*加速。** AWS、Azure、Google Cloudがそれぞれ+28% / 約+40% / +63%という同一四半期の数字は異例だ。直前の2四半期では、少なくとも1社が遅れを取っていた。3社の同時加速は、AIワークロードが概念実証（PoC）段階を越え、実際のクラウド消費へと移行したことを示す最も明確なシグナルだ。

**観察2 — 設備投資は削減ではなく、*増加*している。** Amazon $200B維持、Alphabet $180〜190B、Metaは$125〜145Bへ引き上げ、Microsoft 3Q年率換算で約$128B。ハイパースケーラー株にとってはFCFの圧迫要因だが、サプライチェーンにとっては受注の増加を意味する。

**観察3 — ただし市場は設備投資のROIを厳しく審査している。** 2026年の設備投資ガイダンス引き上げ直後にMetaの株価が下落したことがその典型だ。つまり*設備投資の引き上げそのもの*はハイパースケーラー株を押し上げない。結果として**同じ設備投資の数字が、サプライチェーンには会計上のプラス材料として、ハイパースケーラー自身には会計上の慎重材料として読まれる**という非対称性が生まれる。

## 3. 設備投資の規模 — AIインフラサイクルの強度

4社の2026年AI・クラウド設備投資の年率換算の保守的な試算：

```
Amazon 2026年AI投資目標                    = $200B
Alphabet 2026年設備投資ガイダンス中央値    = $185B   (180〜190B)
Meta 2026年設備投資ガイダンス中央値        = $135B   (125〜145B)
Microsoft 3Q設備投資 × 4（年率換算）       = $127.6B ($31.9B × 4)
─────────────────────────────────────────────
年率換算の代理指標                         ≈ $647.6B  ≈ 約$650B
```

留意点 — これは*厳密な年間設備投資合計ではない*。Microsoftは単一四半期からの年率換算であり変動がある。Amazonの「$200B AI投資」には一部の非クラウド項目が含まれる。この数字は**正確な合計値ではなく、需要の強度を示す代理指標**として扱うべきだ。

それでも示唆は明快だ：

> **ハイパースケーラーにとってこれは約$650BのFCF負担だ。韓国サプライチェーンにとってこれは約$650Bの調達予算だ。**

同じ数字が、サプライチェーンのどちら側にいるかによって、損益計算書上で正反対の意味を持つ。その非対称性がこのサイクルの本質だ。

## 4. サムスン電子 — ビッグテック設備投資の一次受取先

### 4.1 1Q26決算が示したこと

サムスン電子 1Q26 — 売上高 **₩133.9T**、営業利益 **₩57.2T**、うち**DS部門が₩53.7T**を占めた。DS部門の全社営業利益に占める割合：

```
DS比率 = 53.7 / 57.2 = 93.9%
```

利益構造は恒久的に変化した。収益を牽引するのはPC・モバイル向けメモリではなく、**AIサーバーメモリスタック**だ。Reutersは今回の決算をAIインフラのメモリ需要に支えられた過去最高水準の四半期と報じている。

### 4.2 需要の伝達経路はダイレクトだ

```
ハイパースケーラークラウドの成長（+28% / +40% / +63%）
       ↓
AIデータセンター設備投資（年率約$650B）
       ↓
GPU / TPU / ASICサーバーの増設
       ↓
HBM / DDR5 / SOCAMM / eSSD / サーバーDRAM需要
       ↓
サムスン電子 DS売上 + マージン
```

1Q26においてこの経路のどの段階も途絶えていない。AWS / Azure / GCPの同時加速は、これが単発イベントではないことの直接的な証拠だ。

### 4.3 チェックすべき指標

| 確認項目 | 意味 | 現状 |
| --- | --- | --- |
| HBM4 / HBM4E の顧客拡大 | SK hynixとの差を詰められるか | **最重要 — 追加上昇の核心** |
| サーバーDRAM価格 | AIサーバーの供給逼迫を反映 | 良好 |
| eSSD / NAND価格 | AI推論・ストレージのボトルネック波及 | 良好 |
| DS営業マージン | サイクルピークの先行指標 | 依然強い；ピークシグナルなし |
| ファウンドリのAI / HPC受注 | メモリ以外のオプション価値 | 注視中 |

リスクは需要側にあるのではない。(1) メモリ価格ピークのタイミング、(2) HBM顧客多様化のスピード——この2点が懸念材料だ。SK hynixがNVIDIA向けHBMで先行する中、サムスン電子のHBM4認定取得の進捗が今後12カ月の株価における最大の分岐点となる。

### 4.4 出尽くしリスク — 低い

理由は3つ：

1. ハイパースケーラーのクラウド成長が*同時に*加速している。
2. 設備投資は削減ではなく*引き上げ*られている。
3. 1Q26のDS決算がAIメモリ需要を損益計算書レベルで既に実証している。

短期的な利益確定は自然な動きだが、構造的な出尽くりは考えにくい。既存保有者はポジション維持を推奨。新規参入は5日移動平均線か20日移動平均線への押し目、またはメモリ価格の追加データポイント確認後に分散買いが望ましい。

## 5. サムスン電機 — 方向性は正しい、エントリー価格が問題

### 5.1 1Q26決算と構造変化

サムスン電機 1Q26 — 売上高 **₩3.21T**、営業利益 **₩280.6B**。売上高前年比+17%、営業利益+40%。

```
売上高前年比 = +17%
営業利益前年比 = +40%
営業レバレッジ = 40 / 17 ≈ 2.4×
```

2.4倍の営業レバレッジは、**ミックス改善が本物であることを示す最も明確な証左**だ。AIサーバー向けの高スペックMLCCと高層数FC-BGAが全社マージンを引き上げている。サムスン電機が単一のスマートフォンサイクルに縛られなくなったことを示す、初めての会計上のシグナルだ。

### 5.2 需要の伝達経路 — 一段階遅れる

```
ビッグテックの設備投資
       ↓
AIアクセラレーター / ASIC / ネットワークチップの出荷
       ↓
高層FC-BGA / 先端パッケージ基板
AIサーバーの電源・ネットワーク・ストレージ周辺のMLCCコンテンツ
       ↓
サムスン電機 コンポーネント + パッケージソリューション
```

サムスン電子と比べると、経路が**一段階遅れる**。メモリの受注はハイパースケーラーがデータセンターを建設した瞬間に発生する。MLCCとFC-BGA の受注はアクセラレーター / ASIC / ネットワークチップの出荷が確定して初めて動き出す。このタイムラグが、企業業績と株価の反応の乖離を生み出す構造的な原因だ。

### 5.3 サムスン電子より慎重な理由

問題は方向性ではない。**価格**だ。

| 比較軸 | サムスン電子 | サムスン電機 |
| --- | --- | --- |
| ビッグテック設備投資への感応度 | 非常に直接的 | 直接的だが一段階遅れ |
| 業績サプライズの大きさ | 非常に大きい | 堅調だが相対的に限定的 |
| 先取り済みの上値 | あり | **より大きい** |
| バリュエーション正当化の条件 | HBM4顧客拡大 + DS マージンの持続 | **AI MLCC / FC-BGA の2〜3四半期連続の業績上方修正** |
| 出尽くしリスク | 低〜中 | **中〜高** |

市場はすでにサムスン電機の評価軸を「スマートフォン向けMLCCサプライヤー」から「AIサーバー部品メーカー」へと切り替えた。この時点で株価が必要とするのは**上方修正が続くこと**であり、「また好決算だった」という事実ではない。強い1四半期はすでに株価に織り込まれている。

### 5.4 追加上昇を正当化する条件

以下のうち少なくとも2点が実現する必要がある：

- 2Q / 3Qの業績予想が連続で上方修正される。
- AIサーバー向け新規顧客の拡大（具体的なコメントや増産アナウンス）。
- パッケージソリューション部門の営業マージンが10%に接近。
- 車載向けMLCCのさらなるミックス改善。

4月30日の決算で*追加的に*これらの条件が揃うことはなかった。「好決算＋より強いガイダンス」の組み合わせがなければ、現水準での追いかけ買いを正当化するロジックは弱い。

| 投資家区分 | アクション |
| --- | --- |
| **サムスン電子 既存保有者** | ホールド推奨。HBM4の顧客拡大と2QのDSマージンが確認されるまでは減らす理由がない。 |
| **サムスン電子 新規参入希望者** | 上げ相場を追いかけるのは非効率。5日または20日移動平均線への押し目、またはメモリ価格の追加データ確認後に分散買い。 |

要点は明快だ。**同じビッグテック決算が、2つの異なるトレード判断を支持する。** AIの設備投資*サイクル*を取りにいくならサムスン電子を持つ。AI部品の*ナラティブ*を取りにいくなら、価格が来るまで待つ。

## 8. 締めの一言

1Q26のビッグテック決算が明確に伝えるのはただひとつ：**AI需要は曲がっていない、設備投資は引き上げられた、クラウド売上は本物の成長を遂げている。**市場が加えたのは、設備投資ROIへの厳しい審査という要素だ。

この分離が、韓国株においてクリーンな非対称性を生む。設備投資が直接売上に変換されるサムスン電子が有利。先にナラティブが株価に到達したサムスン電機はさらなる実績の積み上げが必要だ。**6〜12カ月の相対選好：サムスン電子 > サムスン電機。** 今四半期もこの原則は変わらない——良い企業と良い投資タイミングは別物だ。
