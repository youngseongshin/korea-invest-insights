## TL;DR

* 현재 중국 AI API 가격은 <strong>구조적으로 낮아진 원가와 전략적 출혈 경쟁이 섞인 가격</strong>이다. 모든 초저가가 유지되기도, 미국 프런티어 수준으로 복귀하기도 어렵다.
* 홍콩 공시가 답을 나눈다. 지푸의 API 매출총이익률은 <strong>18.9%</strong>(전년 3.3%)로 개선돼 <strong>한계 추론원가는 넘기 시작</strong>했다. 그러나 총 매출총이익은 R&D의 <strong>9.3%</strong>에 불과해 <strong>완전원가는 미회수</strong>다.
* 중국의 원가 우위는 실재하지만 화웨이 칩이나 싼 전기가 아니다. 순서는 <strong>모델 구조·활성 파라미터 축소 > 배치·캐싱·양자화 > 클라우드 GPU 이용률 > 낮은 목표 마진 > 전력·국내 NPU</strong>다.
* 전력 효과를 크게 잡아도 컴퓨트 가격의 <strong>약 2%</strong>에 그친다. Qwen 3.5 Flash가 베이징·프랑크푸르트·버지니아에서 같은 가격이라는 사실이 이를 뒷받침한다.
* 전기차와 달리 AI 모델은 <strong>전환비용이 낮다</strong>. 승자독식이 나온다면 모델이 아니라 클라우드·유통 계층이다. 가장 유력한 경로는 <strong>저가 과점(60%)</strong>이다. \[분석 목적\]

## 1. 얼마나 싼가

2026년 7월 기준 100만 토큰당 가격이다. \[Fact: 각사 공식 가격표\]

| 모델 | 입력 | 출력 |
|---|---:|---:|
| DeepSeek V4 Flash | 0.14달러 | 0.28달러 |
| DeepSeek V4 Pro | 0.435달러 | 0.87달러 |
| Gemini 3.5 Flash | 1.50달러 | 9달러 |
| GPT-5.6 Sol | 5달러 | 30달러 |

DeepSeek V4-Pro는 GPT-5.6 Sol보다 입력이 약 <strong>11배</strong>, 출력이 약 <strong>34배</strong> 싸다. 물론 성능, 지연시간, SLA, 보안과 도구 수준까지 동일하다는 뜻은 아니다.

### 중국 안에서도 가격 차이가 크다

- Qwen 3.7 Max: 입력 12위안, 출력 36위안, 현재 50% 할인
- Doubao Seed 2.1 Pro: 입력 6위안, 출력 30위안
- DeepSeek V4-Pro: 중국 경쟁사보다도 훨씬 공격적

따라서 <strong>DeepSeek 가격을 중국 전체의 정상가격으로 보면 안 된다</strong>. \[Inference: 가격 분산 해석\]

## 2. 지속 가능한 부분: 한계원가

DeepSeek V4-Pro는 총 1.6조 파라미터 중 <strong>490억개만 활성화</strong>하는 MoE 구조다. 높은 가동률과 배치 처리, 캐싱을 결합하면 실제 토큰당 연산비용을 크게 낮출 수 있다. 반복 입력의 캐시 히트 가격은 100만 토큰당 불과 <strong>0.003625달러</strong>다. \[Fact: DeepSeek 공식 자료\]

그래서 다음 트래픽은 현재 가격도 한계비용 기준으로 지속 가능할 수 있다.

- 배치 처리
- 캐시 히트
- 지연시간을 보장하지 않는 트래픽
- 높은 가동률을 유지할 수 있는 대량 사용

## 3. 지속하기 어려운 부분: 완전원가

현재 공개가격만으로는 모델 훈련비, R&D, 실패한 실험, 유휴 캐파, 보안·영업·기업용 SLA까지 회수하기 어렵다.

가격 정상화도 이미 시작됐다. \[Fact: 각사 조치 및 보도\]

- DeepSeek는 V4 가격을 75% 낮춘 후 <strong>피크 시간대 가격을 두 배로 인상</strong>
- 알리바바도 일부 AI 서비스 가격을 최대 <strong>34% 인상</strong>
- 기업용 전용 처리량과 저지연 서비스는 공개 API보다 높은 가격 적용

초저가가 모든 시간대와 고객군에서 유지되지는 못한다는 증거다.

| 서비스 | 지속 가능성 |
|---|---|
| 캐시·배치·비우선 트래픽 | 높음 |
| 피크 시간·저지연·고동시성 | 낮음 |
| 기업용 보안·SLA | 별도 고가 요금 정착 |
| 독립 API 사업의 완전원가 회수 | 현재 가격으로는 어려움 |

## 4. 공시로 검증한다: 지푸와 미니맥스

여기부터가 이 글의 핵심이다. 추측 대신 <strong>홍콩 상장사의 공시</strong>로 확인할 수 있다.

| 2025년 | 지푸(02513) | 미니맥스(00100) |
|---|---:|---:|
| 매출 | RMB 7.24억 | 7,904만달러 |
| API·플랫폼 매출 비중 | 26.3% | 32.8% |
| API 또는 전사 매출총이익률 | API 18.9% | 전사 25.4% |
| 전년 매출총이익률 | API 3.3% | 전사 12.2% |
| R&D / 매출 | 439% | 320% |
| 조정순손실 / 매출 | 439% | 317% |
| 매출총이익 / R&D | <strong>9.3%</strong> | <strong>7.9%</strong> |

\[Fact: 지푸 2025 실적, 미니맥스 2025 연차보고서\]

### 지푸: 한계원가 회수는 확인됐다

API·오픈플랫폼 매출은 RMB 4,848만에서 <strong>1.904억으로 293% 증가</strong>했고, 매출총이익률도 3.3%에서 18.9%로 상승했다. 회사는 추론 효율, 규모의 경제, 가격 인상을 원인으로 들었다.

더 중요한 사실이 있다. 2026년 3월 기준 API 가격을 전년 말보다 <strong>83% 올렸는데도 수요가 공급을 웃돌았다</strong>. 현재 GLM-5 공식 가격은 입력 RMB 4, 출력 RMB 18/백만 토큰이다. 상위 API는 직접 추론원가 이하의 덤핑 구조에서는 벗어난 것으로 보인다. \[Inference: 가격 인상과 수요 초과의 함의\]

### 미니맥스: 개연성은 높지만 미확정

API만의 매출총이익률은 공개하지 않았다. 전사 매출총이익률은 12.2%에서 <strong>25.4%</strong>로 개선됐고, 회사는 모델·시스템 효율과 인프라 배분 최적화를 원인으로 제시했다.

현재 M2.5 가격은 입력 0.30달러, 출력 1.20달러/백만 토큰이고 고속형은 0.60달러/2.40달러다. 다만 M2.5는 2026년 출시됐고 공시 손익은 2025년까지이므로, <strong>현재 가격의 API 단독 원가율은 아직 검증되지 않았다</strong>. \[Blocked\]

### 완전원가는 두 회사 모두 미회수

지푸의 총 매출총이익 RMB 2.97억은 R&D RMB 31.80억의 <strong>9.3%</strong>에 불과하다. 미니맥스도 매출총이익 2,008만달러로 R&D 2.53억달러의 <strong>7.9%</strong>만 충당했고, 영업현금 유출은 2.80억달러였다.

미니맥스가 2025년 알리바바그룹에서 구매한 서비스는 <strong>7,588만달러로 매출의 96%</strong>였다. 학습용과 서비스용 클라우드가 섞인 금액이어서 API 원가로 그대로 볼 수는 없지만, 외부 컴퓨팅 인프라 의존도가 높다는 증거다. 알리바바는 미니맥스 지분 17.06%를 보유한다.

중요한 점은 공시가 관련자 거래를 <strong>일반 고객에게 제공되는 공개 가격·조건에 따라 진행했다</strong>고 밝힌다는 것이다. 전략적 관계로 인한 용량 확보와 통합 편의는 인정할 수 있지만, <strong>숨은 가격 보조가 있었다고 단정할 근거는 없다</strong>. \[Inference: 공시 문언 해석\]

상장 후에도 지푸는 HK$313.75억, 미니맥스는 신주와 전환사채로 합계 HK$159.57억을 순조달했다. 조달 자금 대부분이 R&D와 컴퓨팅 인프라로 들어간다. 자금조달 자체가 API 적자의 증거는 아니지만, <strong>프런티어 모델 경쟁이 내부 현금만으로 유지되는 단계는 아니다</strong>. \[Fact: 공시\]

### 판정

1. 유료 상위 API의 한계원가 지속성: 지푸는 <strong>확인</strong>, 미니맥스는 개연성이 높지만 미확정
2. 학습비까지 포함한 완전원가 지속성: 두 회사 모두 <strong>미충족</strong>
3. 가격전쟁 지속 가능성: <strong>높음</strong>. 자본시장 조달로 적자 가격도 오래 유지 가능
4. 시장 구조: 승자독식보다 지푸의 중국 기업·온프레미스, 미니맥스의 해외·소비자·멀티모달처럼 <strong>영역별 과점</strong> 가능성

## 5. 원가 우위 분해: 무엇이 진짜 이유인가

중국의 API 서빙 원가 우위는 실재한다. 다만 핵심은 저렴한 전기나 화웨이 칩 하나가 아니라 다음 조합이다.

```text
작은 활성 파라미터·양자화
+ 높은 배치 처리율·캐시 활용
+ 알리바바 같은 대규모 클라우드의 GPU 이용률
+ 낮은 목표 마진
+ 중국 서부의 저렴한 전력
```

| 요인 | 원가 우위 | 판단 |
|---|---:|---|
| 모델 구조·양자화·캐싱 | 매우 큼 | 중국 API 가격 경쟁력의 핵심 |
| 클라우드 규모·가동률 | 큼 | 알리바바의 가장 확실한 해자 |
| 낮은 목표 마진 | 큼 | 가격이 싼 이유지만 장기 수익성에는 약점 |
| 화웨이 Ascend | 중간, 조건부 | 최적화된 워크로드에서는 유리, 범용성은 낮음 |
| 중국 서부 전력 | 중간 | 배치형 추론에 유효, 글로벌 실시간 API에는 제한 |
| 정부 보조·정책금융 | 존재 | 산업 육성에는 유리하지만 서비스별 금액은 불투명 |

## 6. 전력은 정말 결정적인가

중국 전력이 싼 것은 사실이다. 닝샤 중웨이 데이터센터의 도착 전력가격은 2026년 기준 <strong>0.36위안/kWh</strong>로 중국 동부의 약 45% 수준이다. 미국의 2025년 산업용 평균은 <strong>8.62센트/kWh</strong>로, 환율 7.2위안/달러 가정 시 약 0.62위안/kWh다. 중웨이가 약 <strong>42% 저렴</strong>하다. \[Fact: 중국 지방정부 자료, 미국 EIA\]

그러나 전력만으로 API 가격이 5배, 10배 싸지지는 않는다. 전력효과를 <strong>크게 잡아</strong> 카드당 IT 부하 1kW, PUE 1.1로 가정하면 이렇다.

```text
중웨이:   1kW × 1.1 × 0.36위안 = 0.40위안/시간
미국 평균: 1kW × 1.1 × 0.62위안 = 0.68위안/시간
차이:                            약 0.29위안/시간
```

알리바바가 제시하는 L20 온디맨드 가격이 14.4위안/시간이므로, 이 상한 가정에서도 전력 차이는 컴퓨트 가격의 <strong>약 2%</strong>다. \[Inference: 자체 계산\]

전력이 중요하지만 API 가격 격차의 주범은 아니라는 뜻이다.

### 결정적 반증

서부 전력 우위는 배치 추론·학습에는 잘 맞지만, 저지연 API는 사용자와 가까운 동부·해외 리전에 배치해야 한다. 그리고 알리바바의 Qwen 3.5 Flash 가격은 <strong>베이징뿐 아니라 프랑크푸르트와 버지니아에서도 입력 0.029달러, 출력 0.287달러/백만 토큰으로 동일</strong>하다. \[Fact: Alibaba Model Studio 가격표\]

저가 API가 중국 전력만의 결과가 아님을 보여주는 가장 직접적인 증거다. \[Inference: 리전 동일가격의 함의\]

## 7. 알리바바의 진짜 해자는 이용률

알리바바 클라우드는 전력보다 <strong>GPU를 쉬지 않게 만드는 능력</strong>에서 강하다. \[Fact: Alibaba 공식 문서\]

- 배치 추론: 정가 대비 <strong>50% 할인</strong>
- L20 스팟 인스턴스: 온디맨드 14.4위안에서 통상 2.88위안 부근, 약 <strong>80% 절감</strong>
- 유휴 GPU: 활성 단가 0.000018달러/CU 대비 유휴 단가 0.000007달러/CU
- 온디맨드, 스팟, 오토스케일링을 혼합해 피크 트래픽 대응

다만 스팟은 회수 위험이 있어 SLA가 엄격한 실시간 API 전체를 스팟으로 돌릴 수 없다. 기본 수요는 온디맨드나 전용 자원에 두고, 탄력 수요만 스팟으로 처리하는 구조다. \[Inference: 운영 제약\]

## 8. 화웨이 Ascend는 무조건 싼가

엄밀히 말하면 Ascend는 GPU가 아니라 <strong>NPU</strong>다.

화웨이는 CloudMatrix 384에서 다음을 주장한다. \[Fact: 화웨이 공식자료\]

- 단일 카드 환산 추론 처리량 2,300 tokens/s
- 기존 비슈퍼노드 대비 약 4배
- MFU 50% 이상 개선
- H20 대비 카드당 평균 추론성능 3-4배

자원 풀링과 MoE 전문가 병렬화로 저사양 개별 칩의 약점을 시스템 설계로 보완하는 방식이다.

### 그러나 곧바로 낮은 원가로 읽으면 안 된다

- 화웨이는 <strong>동일 모델·동일 지연시간·동일 전력 기준의 외부 검증 TCO를 공개하지 않았다</strong>. \[Blocked\]
- CloudMatrix는 많은 NPU와 네트워크를 묶어 처리량을 높이는 구조이므로 <strong>카드당 처리량과 클러스터 전체 비용은 다르다</strong>.
- 2026년 Ascend 910 현장 연구에서는 안정적 추론을 위해 소스 패치 12개, 일부 고처리량 기능 비활성화, 반복 장애 대응 장치가 필요했다. 하드웨어 가격이 싸더라도 <strong>엔지니어링과 운영비가 올라갈 수 있다</strong>. \[Fact: Ascend 현장 연구\]

## 9. 전기차와 무엇이 다른가

비슷한 점은 전략산업, 대규모 투자, 대기업 보조, 가격 인하를 통한 점유율 확보다.

차이는 <strong>AI 모델의 전환비용이 훨씬 낮다</strong>는 점이다. API 규격이 비슷하고, 고객은 여러 모델을 동시에 라우팅하거나 오픈소스 모델을 자체 운영할 수 있다. 한 사업자가 가격을 올리면 경쟁 모델과 자체 호스팅이 가격 상한을 만든다.

반면 클라우드, 데이터, 보안, 결제, 광고, 업무도구까지 결합된 플랫폼은 전환비용이 높다. <strong>승자독식이 나타난다면 모델보다 유통과 클라우드 계층</strong>이다. \[Inference: 전환비용 구조\]

가장 가능성 높은 구조는 이렇다.

- 알리바바: 클라우드·전자상거래·기업 고객
- 바이트댄스: 소비자 트래픽·광고·콘텐츠
- 텐센트: WeChat·기업 서비스
- 화웨이·바이두: 국산 인프라와 공공·기업 고객
- DeepSeek 등 독립 모델사 1-2곳: 기술 기준가격 제공

## 11. 반도체 시사점

저렴한 API는 추론 사용량을 늘려 서버 DRAM, NAND, 네트워크와 전력 수요에 유리하다. 반면 토큰당 연산량과 HBM 탑재 강도를 낮춰 프리미엄 GPU와 HBM의 단위당 경제성에는 압박을 준다.

| 대상 | 판단 |
|---|---|
| 삼성전자 | 범용 DRAM·NAND 비중이 높아 상대적으로 완충 |
| SK하이닉스 | HBM 희소성 프리미엄에 장기 경고 |
| 엔비디아 | 프리미엄 GPU 단가에는 부담, 총 추론량 증가는 방어 요인 |
| 알리바바·텐센트 | API 마진보다 클라우드·유통 생태계 확장의 수혜 |

핵심은 API 가격 하락률보다 <strong>전체 토큰 사용량 증가율</strong>이다. 사용량이 가격 하락보다 빠르게 늘면 제본스 효과가 이기고, 그렇지 않으면 프리미엄 GPU와 HBM부터 멀티플이 낮아진다. \[Inference: 방향 판단\]

## 12. 결정적 검증은 2026년 상반기 공시다

- <strong>지푸의 API 매출총이익률</strong>이 83% 가격 인상 후 25-30% 이상으로 올라가는가
- <strong>미니맥스가 API 원가율을 공개</strong>하는가
- <strong>매출 증가율이 컴퓨팅비 증가율을 계속 웃도는가</strong>
- 피크·SLA 요금과 공개 API 요금의 격차가 굳어지는가
- 화웨이가 동일 조건 외부 검증 TCO를 공개하는가

이 다섯 가지가 "지속 가능한 원가"와 "자본으로 버티는 가격"을 가른다.

## 마무리

중국 API 가격이 낮은 것은 <strong>이미 자립 가능한 정상가격이라서가 아니다</strong>. 추론 효율 개선과 막대한 자본조달이 함께 받치기 때문이다.

중국은 국내·배치형·최적화된 모델 서빙에서 구조적 원가 우위가 있다. 그러나 현재 관찰되는 극단적으로 낮은 API 가격을 화웨이 칩과 싼 전력만으로 설명할 수는 없다. 가장 큰 원가 우위는 모델 크기와 활성 파라미터 축소, 배치·캐싱·양자화, 클라우드 GPU 이용률, 낮은 목표 마진 순이며 전력과 국내 NPU는 그 다음이다.

투자 관점에서 중국 저가 API가 곧바로 "중국 AI가 압도적인 비용 해자를 확보했다"는 뜻은 아니다. 더 정확한 해석은 <strong>추론 서비스가 빠르게 커머디티화되면서 모델 회사의 마진은 낮아지고, 클라우드·전력·반도체 인프라 쪽으로 가치가 이동할 가능성이 커진다</strong>는 것이다.

이는 서구 모델 사업자의 API 마진에는 부담이지만, 조달금이 다시 학습과 컴퓨팅 인프라에 투입되므로 <strong>반도체 총수요 감소를 뜻하지는 않는다</strong>.
