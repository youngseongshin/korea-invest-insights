*5월 15일 코스피가 -6.12% 빠졌다. 사상 최고치 8,000을 돌파한 직후의 폭락. 코스닥은 -5.14%. 매도 사이드카가 발동됐다. 이런 날 투자자들은 두 가지 유혹을 받는다. 하나는 "안 빠진 종목"을 찾아 다음 주도주를 골라내는 것. 다른 하나는 "다 빠졌으니 싸다"며 추격 매수하는 것. 둘 다 위험하다. 진짜 봐야 할 것은 종목이 아니라 매크로 게이트다 — 미국 장기금리, 유가, 원/달러, VIX. 이 네 가지가 안정되기 전에는 어떤 좋은 종목도 추격 매수는 비효율이다.*

---

## 핵심 요약

* **5월 15일 한국 시장**: 코스피 7,493.18(**-6.12%**), 코스닥 1,129.82(**-5.14%**). 사상 최고치 돌파 직후 매도 사이드카 발동.
* **급락의 원인**: 단순 차익실현이 아님. 미국 PPI +6.0%(2022년 이후 최대), 일본 PPI +4.9%, JGB 10년 1997년 이후 최고, 미국 10년 4.46%, 브렌트유 108달러, 원/달러 1,500원 근접 — **글로벌 매크로 사이클이 위험자산을 동시에 압박**.
* **상대강도(RS) 종목들이 있다**: 하나마이크론 +18.6%, 제주반도체 +8.9%, SFA반도체 +4.0%, 삼성전기 -1.4%(지수 대비 +4.7%p). 두산로보틱스 +19.3%, 로보스타 +9.9%.
* **그런데 추격하면 안 된다**: '안 빠진 종목 = 다음 주도주'가 아님. 매크로가 안 풀리면 RS 종목도 다음 날 따라 빠질 수 있음.
* **진짜 봐야 할 것 — 매크로 게이트**: 미국 10년 4.45% 하회, 브렌트 105달러 하회, 원/달러 1,480원 하회, VIX 18 하회. 네 개 중 2\~3개가 안정돼야 신규 매수 의미.
* **오늘 RS 스크리닝의 정확한 용도**: '매수 신호'가 아니라 **'감시 리스트 생성 신호'**.

## 1. 무슨 일이 벌어졌나

### 1.1 한국 시장

```

코스닥: 1,129.82 (-5.14%)

특이사항:
- 장중 한 때 사상 최고치 돌파 후 급반락
- 매도 사이드카(주가 급락 시 프로그램 매도 5분 정지) 발동
- 외국인·기관 대규모 순매도, 개인이 받음

이건 단순 '차익실현'으로 보기 어려운 규모.
하루 6% 하락은 2024년 8월 이후 최대.

```
같은 날 글로벌 시장:
- 미국 나스닥 선물 급락
- 일본 닛케이도 하락
- 중국 본토는 회담 결과 실망 매도

## 2. 이런 날 상대강도(RS) 종목 찾기의 함정

### 2.1 상대강도가 뭔가

```
상대강도(Relative Strength, RS):
종목 등락률 - 시장 등락률

예: 코스피 -6.12% 빠진 날 어떤 종목이 +5%면
→ RS = +5% - (-6.12%) = +11.12%p
→ "지수보다 11.12%p 강했다"

왜 보는가:
→ 시장이 빠질 때 안 빠지는 종목 = 매수세가 강한 종목
→ 시장이 회복하면 가장 먼저 오를 가능성
→ 다음 주도주 후보 발굴 도구
```

### 2.3 함정 1 — '상한가 급등형'을 '실적 강세'와 구분 못 함

```
RS 상위에는 두 종류의 종목이 섞여 있다:

A. 실적 기반 강세 (좋은 RS):
   - 하나마이크론: 1Q26 영업이익 720억 (+513.6%)
   - 제주반도체: 1Q26 영업이익 671억 (+1,713%)
   - 삼성전기: MLCC 가격 인상 + AI 부품 thesis
   → 펀더멘털 뒷받침

B. 테마·뉴스 급등 (위험한 RS):
   - 에스에이엠티 상한가 +30%
   - 삼지전자 상한가 +30%
   - 한선엔지니어링 +19%
   → 특정 뉴스 또는 수급 이벤트

상한가 종목들을 'RS 1\~2위'로 잡으면
→ 다음날 갭다운 후 음봉 가능성 높음
→ 진짜 주도주 후보가 아님
```

### 2.4 함정 2 — '종가 위치'를 안 보면 다 같은 종목

```
종가 위치 = (종가 - 저가) / (고가 - 저가)

같은 +5% 상승이라도:
- 종가 위치 90% → 장 마감까지 매수세 유지 (좋은 신호)
- 종가 위치 20% → 장중 고점에서 크게 밀림 (분배성 매물)

오늘 데이터로 보면:

종목         등락률     종가 위치    해석
하나마이크론    +18.6%     71.7%      ★ 매수세 장 후반까지 유지
제주반도체     +8.9%      62.2%      좋지만 투자경고 리스크
SFA반도체     +4.0%      19.5%      ✗ 장중 강하게 밀림 (RS 품질 낮음)
삼성전기      -1.4%      19.6%      ✗ 지수 대비 방어는 됐지만
                                    장중 분배성 매물 출회

→ 단순 등락률만 보면 SFA반도체가 '강한 종목' 같지만
→ 종가 위치를 보면 매수보다 매도가 우세였다는 뜻
→ '같은 RS 4위'라도 품질이 다르다
```

### 2.5 함정 3 — 매크로가 안 풀리면 RS 종목도 다음날 따라 빠진다

```
역사적 패턴:
지수 급락일에 RS 1\~2위 종목은
다음 거래일에 두 갈래로 갈린다.

A. 시나리오 (매크로 회복):
   → 지수 반등
   → RS 종목이 선도 상승 (+10\~15% 가능)
   → 새로운 주도주로 자리잡음

B. 시나리오 (매크로 추가 악화):
   → 지수 추가 하락
   → RS 종목도 추격 매도 발생
   → 어제 강했던 종목이 -10\~15% 되돌림
   → 단기 트레이더가 손절

핵심: A와 B를 결정하는 건 종목이 아니라 매크로
→ 종목 분석 전에 매크로를 봐야 한다
```

## 3. 매크로 게이트 — 월요일에 진짜 봐야 할 것

### 3.1 게이트 1: 미국 장기금리

```
현재 상태:
미국 10년: 4.46%
미국 30년: 5.02%

회복 신호:
미국 10년 4.45% 하회 (1차)
미국 30년 5.0% 하회 (강한 회복)

왜 중요한가:

## 4. 매크로 게이트 기반 매매 프레임

### 4.1 게이트 통과 조건

```
신규 매수 강도를 높이는 조건:
4개 게이트 중 최소 2\~3개가 안정

구체적으로:
1. 미국 10년 4.45% 아래로 하락
2. 브렌트유 105달러 아래로 하락
3. 원/달러 1,480원 아래로 안정
4. VIX 18 아래로 하락
5. 코스피/코스닥 전일 저점 이탈 없이 양봉 마감

이 중 최소 2\~3개 충족 → 부분 매수 가능
1개 이하 충족 → 매수 금지
```

### 4.2 시나리오별 행동

| 시나리오 | 조건 | 행동 |
| --- | --- | --- |
| **A. 매크로 회복** | 4개 게이트 중 3개 이상 안정 | RS 종목 분할 매수 검토. 단, 최대 사이즈의 30\~40% |
| **B. 매크로 중립** | 1\~2개 게이트만 안정 | 종가만 확인. 신규 매수 최소화 |
| **C. 매크로 악화** | 게이트 추가 악화 | 신규 매수 금지. 보유 종목은 thesis 훼손 여부로만 판단 |

### 4.3 'RS 종목 매수'와 '매크로 게이트'의 관계

```
잘못된 순서:
RS 종목 발굴 → 매수 → 매크로 확인

올바른 순서:
매크로 게이트 확인 → 통과 시 RS 종목 선별 → 추격 매수

이유:
- 매크로가 안 풀리면 RS 종목도 따라 빠진다
- 좋은 종목을 비싼 가격에 사면 결과가 나쁘다
- 진짜 주도주는 한 번에 만들어지지 않는다 — 며칠\~몇 주 강세 지속 후 확인됨
```

## 5. 만약 매크로가 회복된다면 — 종목별 우선순위

게이트가 통과되면 이런 순서로 보면 된다.

### 5.1 1순위 — 하나마이크론

```
펀더멘털:
1Q26 매출 5,077억(+62.8% YoY)
영업이익 720억(+513.6% YoY)
이익률 14.2%
컨센서스 영업이익 +29.7% 상회

종가 위치: 71.7% (강한 매수세 유지)
거래대금: 6,888억원 (충분)

판단:
- 실적 기반 RS의 가장 깨끗한 후보
- 다만 한국투자증권 목표가 55,000원에 근접 (현재 52,900원, 업사이드 +4%)
- 추격보다 눌림 또는 거래대금 유지 확인 후

## 7. 핵심 교훈

### 7.1 '안 빠진 종목 = 다음 주도주'가 아니다

```
잘못된 추론:
"오늘 시장이 빠졌는데 이 종목은 안 빠졌다"
→ "강하다"
→ "내일도 강할 것"
→ "사야 한다"

올바른 추론:
"오늘 시장이 빠졌는데 이 종목은 안 빠졌다"
→ "오늘 기준으로 매수세가 있었다"
→ "내일도 강하려면 매크로가 안정되어야 한다"
→ "매크로 확인 후 거래대금 유지되면 매수 후보"
```

### 7.2 주도주는 '한 번'이 아니라 '여러 번' 강한 것

```
하루 강한 종목 ≠ 주도주
일주일 강한 종목 = 후보
한 달 강한 종목 = 주도주

오늘 RS 1위라고 '내일 주도주'가 되는 게 아니다.
다음 5\~10거래일 동안 지수 대비 꾸준히 강하면
그때 주도주로 자리잡는다.

→ 오늘 매수는 '주도주 매수'가 아니라 '주도주 후보 베팅'
→ 매크로가 안 풀리면 후보들도 다 떨어진다
```

### 7.3 종목 분석 < 매크로 게이트

```
이 글의 핵심 메시지:
"종목보다 사이클이 먼저다"

좋은 종목이 비싼 가격에 잘못된 타이밍에 잡히면
결과는 나쁘다.
평범한 종목이라도 매크로가 회복되는 시점에 잡으면
결과는 괜찮다.

→ 종목 발굴에 쓰는 에너지의 30%만으로
→ 매크로 게이트 4개를 매일 체크하면
→ 매수 타이밍 품질이 압도적으로 좋아진다
```

## 9. 마지막 한 줄

5월 15일 코스피 -6.12%는 단순 차익실현이 아니다. **글로벌 매크로 사이클이 한국 시장에서 결산된 날**이다. 미국 PPI +6%, 일본 PPI +4.9%, 미국 10년 4.46%, 브렌트유 108달러, 원/달러 1,500원 근접 — 이 모든 변수가 같은 방향으로 한국 위험자산을 압박했다.

이런 날에는 두 가지 유혹이 있다. 하나는 "안 빠진 종목"을 찾아 다음 주도주를 발굴하는 것. 다른 하나는 "다 빠졌으니 싸다"며 추격 매수하는 것. **둘 다 매크로 게이트가 안 풀리면 비효율이다.**

월요일에 가장 먼저 봐야 할 것은 하나마이크론도, 삼성전기도, 제주반도체도 아니다. **미국 10년 금리, 브렌트유, 원/달러, VIX** — 이 네 가지다. 이 중 최소 2\~3개가 안정되지 않으면 어떤 좋은 종목도 추격 매수는 비효율이다.

오늘 RS 스크리닝은 '매수 신호'가 아니라 **'감시 리스트 생성 신호'**다. 다음 주도주는 매크로가 회복된 뒤 새로 찾는 게 아니라, **매크로가 나쁠 때도 안 빠진 종목을 기록해뒀다가, 매크로가 회복될 때 사는 것**이다.

**종목보다 사이클이 먼저다. 종목 분석 전에 매크로 게이트부터 보자.**
