- 多头仓位策略：**持有。不加仓。不减仓。** 今天是第一支撑位的确认，而非第二阶段突破。

## 5. 为何"散户 → 机构换手"至关重要

通常的解读框架是"股价上涨是因为外资买入"。但今天并非如此。今天呈现的是更纯粹的多头信号：**边际卖方是散户 / 空头 / 外资券商，边际买方是国内机构。**

为何这比纯外资买入拉升更健康：

| 因素 | 外资主导拉升 | 今日结构 |
|---|---|---|
| 买方构成 | 单一来源（对汇率及宏观敏感） | 国内机构（粘性更强） |
| 可逆性 | 汇率/风险偏好逆转即撤 | 需基本面恶化才会逆转 |
| 筹码质量 | 热钱 | 重新锚定的稳定持股 |
| 卖空格局 | 往往增强 | 趋于减弱（成本端受压） |
| 支撑持久性 | "阻力式支撑"——依赖其不离场 | 基于信念的支撑 |

Kiwoom净流出-62,704股颇具说明意义。Kiwoom客群偏向散户 / 短线 / 信用融资流。**在Kiwoom大量抛售的情况下，价格仍守住₩60,000，意味着弱手离场、强手在该价位承接了筹码。** 这正是支撑底部形成的内在逻辑。

结合新韩证券+51,245 / 韩国投资证券+20,419的净买入，图景更加清晰：**买盘来自机构零售/高净值客户券商窗口，而非单一外资席位。** 多来源国内买盘在结构上比单一来源外资买盘更具粘性。

## 6. 盘口质量评估

### 积极因素

1. **收盘站上₩60,000。** 今天最重要的条件。₩60,000不再只是阻力位——它现在是第一支撑候选位。
2. **收盘₩60,200 vs 卖空VWAP₩60,390。** 明日若站上₩60,400，今日空头立即被套。
3. **在Kiwoom净卖出的情况下价格守住。** Kiwoom净流出-62,704股，大概率是散户 / 日内交易者 / 信用融资抛压。在此压力下守住₩60,000是一次体力测试。
4. **新韩证券 / 韩国投资证券净买入。** +51,245 / +20,419。这是守住₩60,000的主动买盘。
5. **卖空力度减弱。** 今日卖空量约为近10日均值的一半。并非4月23日式的大规模空袭。

### 不足之处

1. **正常交易时段未能收复₩61,700。** 真正的趋势转变应在午后再创当日新高。
2. **未能延续盘前₩62,600高点。** 盘前的积极情绪未能完全延伸至正常交易时段。
3. **成交额未过₩1,000亿。** ₩842.1亿（含NXT）表现尚可，但未达到"明确支撑确认"的₩1,000亿门槛。
4. **外资券商盘口未翻转为净买入。** 估算净流出-38,336股。外资尚未主导行情。
