## 1. 規模——62.5兆ウォンが実際に意味するもの

### 1.1 ボーナス原資の積み上げ方

```
SKハイニックス：
- 2026E営業利益コンセンサス：     227.8兆ウォン
- ボーナス比率：                   営業利益の10%
- 計算：                           227.8兆 × 10% = 22.8兆
- 従業員1人あたり（平均）：        22.8兆 / 約35,000人 = 約6億5000万ウォン

サムスン電子（DSセグメント、条件付き）：
- 2026E DS営業利益コンセンサス：   約331.3兆ウォン
- 特別賞与比率：                   営業利益の12%（検討中）
- 計算：                           331.3兆 × 12% = 39.8兆
- 従業員1人あたり（平均）：        39.8兆 / 約78,669人 = 約5億1000万ウォン

合計：22.8兆 + 39.8兆 = 62.5兆

サニティチェック：
- 従業員1人あたり平均約5億1000万〜6億5000万ウォン（税引き前）
- SKハイニックス2025年の1億5000万ウォンの約4倍
- 韓国の平均世帯年収（約6800万ウォン）の7〜10倍
```

### 1.2 GDPおよび家計消費との比較

```
韓国2025年名目GDP：             約2,663.3兆ウォン
家計消費（46.6%）：              約1,241.1兆ウォン

62.5兆ウォンの相対的な重み：
- 名目GDP比：                    62.5 / 2,663.3 = 2.35%
- 家計消費比：                   62.5 / 1,241.1 = 5.04%
- 2025年補正予算（31.8兆ウォン）比：約1.97倍

読み解き：
- 補正予算規模の民間現金移転
- 意味のあるマクロ変数
- ただし「意味がある」≠「消費回復」
```

### 1.3 国内消費に実際に届く金額はずっと小さい

```
計算式：
国内消費への波及効果 = ボーナス総額
                    × (1 − 実効税率)
                    × 税引き後の限界消費性向
                    × 国内消費比率
                    × 二次乗数

シナリオ別：

ベア（悲観）：
62.5 × (1−45%) × 15% × 60% × 1.1 = 約3.4兆ウォン
GDP比：0.13%

ベース（最も現実的）：
62.5 × (1−42.5%) × 25% × 70% × 1.2 = 約7.6兆ウォン
GDP比：0.28%

ブル（楽観）：
62.5 × (1−40%) × 35% × 80% × 1.3 = 約13.7兆ウォン
GDP比：0.51%

→ 最も現実的な推計：7.6兆ウォン
→ 家計消費への押し上げ効果：+0.61%
→ 意味はあるが、「回復」ではない
```

### 1.4 限界消費性向が低い理由

```
高所得者が追加で1億ウォンを受け取った場合の典型的な使途：

1. 税金：約42〜45%（4200〜4500万ウォン）
   → 政府に流れる

2. 借金返済／貯蓄：約20〜30%（1500〜3000万ウォン）
   → 消費ではなく、負債削減

3. 資産投資：約25〜35%（2500〜3500万ウォン）
   → 株式、不動産、債券に流入
   → 「資産」であって「消費」ではない

4. 実際の消費：約10〜25%（1000〜2500万ウォン）
   - うちプレミアム消費：約70%
   - うち一般消費：約30%

5. 海外消費／国外流出：約5〜10%
   - 旅行、輸入品ラグジュアリー、海外株式

比較（低所得者）：
- 税金：約10〜15%
- 貯蓄：約5〜10%
- 消費：約70〜85%
→ 限界消費性向：約70〜85%

→ 同じ1億ウォンでも受け取る手が違えば、
   国内消費への波及効果は3〜5倍異なる。
```

チャネル1：医学系 → 半導体

   他の韓国産業から人材が流出するという事態が同時進行。

## 3. ボーナスより大きな3つのマクロ変数

### 3.1 変数1——工場設備投資（10年累計で1,000兆ウォン超）

```
ボーナス：年間約62.5兆ウォン。
工場設備投資：10年累計で1,000兆ウォン超。
→ 約16倍大きい。

これが真のマクロ変数だ。

一次効果：工場建設そのもの
- 建設／EPC：約200兆ウォン
- 装置／材料：年間数兆ウォン規模の受注
- ガス／化学品／物流

二次効果：インフラ（約50%は政府が共同負担）
- 電力：変圧器、ケーブル、ESS、発電
- 水：上下水道、処理
- 道路：工業団地アクセス、地域幹線道路
- 住宅：従業員と請負業者向け

三次効果：地域経済
- 地域消費（飲食、医療、教育）
- 地域不動産
- 地域雇用

本質的な読み解き：
この資本は「消費」ではなく「資本蓄積」に流れる。
→ 短期的な消費効果は小さい
→ しかし長期的な韓国経済の構造を変える。
```

### 3.2 変数2——税収＋再分配

```
両社からの追加税収：

直接：
- 法人税：営業利益ブームによる増収 → 100兆ウォン超
- 所得税：ボーナスブームによる増収 → 約25〜31兆ウォン
- 消費税：消費増による増収 → 約5〜10兆ウォン
- 合計：約130〜140兆ウォン

その使途：

シナリオA：産業／インフラへの再投資
- 送電網の拡充
- AI／R&D支援
- 地域インフラ
- 効果：資本蓄積、長期成長

シナリオB：再分配
- 低所得者への移転
- 若者・家族支援
- 地域消費クーポン
- 効果：短期的な消費押し上げが強い

最も現実的な配分：約60% A＋約40% B
→ 両方の効果が発動
→ Bを通じた短期消費効果が大きい。

示唆：
- ボーナス単独の限界消費性向は約25%
- 低所得層への再分配の限界消費性向は約70〜85%
- 同じ金額でも3〜4倍の効果の差
→ 「間接的な再分配」がより強力な消費チャネル。
```

### 3.3 変数3——資本市場の資産効果

```
KOSPIへのインパクト：

サムスン電子：      +125.6%
SKハイニックス：    +179.4%

直接的な資産効果：
- 韓国における株式直接保有世帯：約30%
- 米国（約58%）より低い
- 直接効果はそれゆえ限定的

しかし間接チャネルは強い：
- 国民年金／企業年金の資産増加
- ETF／年金残高の増加
- 金融資産評価額の上昇
- 取引高増加 → 証券会社の利益増加

キウム証券2026年1Q：
- 営業利益：6,212億ウォン（+90.9%）

## 5. 市場に蔓延する4つの誤解

### 5.1 #1——「ボーナス＝消費回復」

```
誤った推論：
「62.5兆ウォン＝消費復活」

現実：
- ヘッドライン：62.5兆ウォン
- 税引き後：約36兆ウォン
- 実際の消費：約7.6兆ウォン（ベースケース）
- GDP比：0.28%

→ 「意味のあるマクロ変数」と「消費回復」は同義ではない
→ 補完要素であって、代替要素ではない。

真の回復に必要なもの：
- 家計債務の軽減
- 不動産の正常化
- 中小企業・自営業者の回復
- 雇用の質の向上
- 少子化の改善

→ 62兆ウォンのボーナスでこれらすべては解決できない。
```

### 5.2 #2——「受益者が全国消費を牽引する」

```

- SKハイニックス：約35,000人

### 5.3 #3——「トリクルダウンが経済全体を押し上げる」

- ボーナス → 不動産と株式（資産）

→ 再分配の方がトリクルダウンより効果的

## 9. 一行の結論

サムスン電子＋SKハイニックスの合算ボーナス原資62.5兆ウォンは、補正予算規模の民間現金移転——名目GDPの2.35%、家計消費の5.04%——だ。これは意味のあるマクロ変数だ。**しかし「意味がある」は「消費回復」を意味しない。** 国内消費への実質的な波及効果のベースケースは約7.6兆ウォン、GDP比0.28%に過ぎない。理由は明快だ：受益者は約11万人（人口の0.23%）、限界消費性向は約25%、大半の資金は不動産、株式、海外消費に吸収される。

**真の変化は「平均値の回復」ではなく「Kシェイプ・リセット」だ。** 韓国の資本市場は「AIメモリETF」（KOSPIの44%）に収束し、労働市場は新たなプレステージ序列に再編され、不動産は「通勤圏＋江南」と「それ以外」に分裂し、消費はラグジュアリーとバリューに二極化し、地理的には「京畿道南部＋忠清北道のAIベルト」とその他へと再形成される。真っ先に消えるのは中間市場だ。

**より大きなマクロ変数は別のところにある。** ボーナス（年間約62.5兆ウォン）は氷山の一角に過ぎない。①工場設備投資（10年累計で1,000兆ウォン超）はボーナスの約16倍規模；②追加税収（年間約130〜140兆ウォン）が再分配されれば限界消費性向は3〜4倍に高まる；③資本市場の資産効果は証券会社の出来高に直結する。**ボーナスは「消費イベント」であり、設備投資・税収・資本市場は「構造イベント」だ。**

**VCと株式投資家にとって、リアルなアルファはKの両端と、その間をつなぐインフラにある。** マス・アフルエントWealthTech（新規富裕層5〜15万世帯）、PropTech 2.0（通勤圏）、AIインフラ隣接SaaS、法務／税務／相続自動化——この4つが最も実行可能なテーゼだ。上場株式での優先順位は、キウム証券（資本市場ベータ）、工場インフラセカンドティア（Hanyang Eng、GST）、現代百貨店（プレミアム消費）、そして押し目での電力インフラだ。

**最大のリスクはAIキャペックスサイクルの変曲点だ。** 2027〜2028年の減速に備えて、サイクル非依存のB2B SaaSとKの下端（バリュー層、サブスクリプション）に10〜15%のヘッジを積んでおく。**「平均に賭けると負ける。Kの両端と、その間をつなぐインフラに賭けよ——それが答えだ。」** この差は1四半期のリターンを決めるのではなく、5年の複利リターンを決める。
