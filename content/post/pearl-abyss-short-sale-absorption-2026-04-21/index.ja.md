## TL;DR

64,362株の空売りが入ったにもかかわらず、終値は当日の空売り平均価格を下回れなかった。空売り平均価格はKRW **54,245**、Pearl Abyssの終値はKRW **54,400**。本日新規に空売りを建てたポジションはすべて、終値時点で含み損となっている。

外国人（+28,741株）と機関（+7,960株）が買いを入れ、空売りフローに対して合計**+36,701株**を吸収した。当日の値動きは安値KRW 53,300から終値KRW 54,400へ、安値からの回復率**+2.06%**。一言で言えば：**売り浴びせたが、崩れなかった。**

フローの全体像は、1日分のデータより重要だ。空売り数量は**4月16日に298,379株**でピークを付け、その後100,764 → 86,097 → **64,362**と急減。ピークから4営業日で**-78.4%の下落**だ。空売りは継続して入ってきている — しかし、強く押せば押すほど、株価は動かなくなっている。

## 2. 値動きを読む

### 2.1 「空売りが強かった日」ではなく「空売りが弱々しく入ってきた日」

64,362株はゼロではない — しかし、Pearl Abyssが最近経験した10万〜30万株規模の空売り圧力と比べれば、本日の強度は明らかに低い。数量の大小より重要なのは**結果**だ：

- 空売り平均価格：**KRW 54,245**
- 終値：**KRW 54,400**
- 結果：**終値が空売り平均を上回った**

意味するところ：**空売りは執行されたが、終値を自分たちの平均価格以下に引き下げられなかった。** 本日の空売りは、相場の主導的な売り圧力ではなく、**調整局面における二次的な売り圧力**として読むべきだ。

### 2.2 空売り以上に雄弁だったプライスアクション

日中の値動き：

```
始値 53,700 → 安値 53,300 → 終値 54,400
```

安値から終値への回復：**+KRW 1,100**、すなわち**+2.06%**：

```
(54,400 − 53,300) ÷ 53,300 = 2.06%
```

加えて、外国人＋機関の合計ネット買いは**+36,701株**。メッセージは明確だ：

> **吸収量が売り＋空売りを上回った。**

### 2.3 アップティック免除比率は戦術的ではなく取引的な性格

本日のアップティック免除比率14.0%は、直近の高圧力日と比べて低水準だ。その性格は**組織的な売り崩しではなく、ヘッジ・取引目的**に近い。

## 4. 10日間ウィンドウの構造的分析

### 4.1 4月16日ピーク後、空売り強度は急落した

空売り数量の推移：

```
4/8  98,973
4/9  232,474
4/10 121,743
4/13 266,406
4/14 115,298
4/15 112,944
4/16 298,379   ← 10日間ピーク
4/17 100,764
4/20  86,097
4/21  64,362
```

重要なのは**4/16が空売りのピーク**だったということだ。その後の推移：

```
298,379 → 100,764 → 86,097 → 64,362
```

ピークからの下落：

```
Δ = 298,379 − 64,362 = 234,017株
%  = 234,017 ÷ 298,379 = 78.4%
```

直近の板を**「空売りが再加速している」**と読むのは誤りだ。正確な読みは**「4月16日の圧力後、空売り強度は急速に冷却している」**だ。

### 4.2 空売り大量流入日と空売り優勢日は別物

- **4/13**：266,406株の空売り、終値は当日の空売り平均を**上回った**。
- **4/16**：298,379株の空売り、終値は当日の空売り平均を**下回った**。
- **4/21**：64,362株の空売り、終値は当日の空売り平均を**上回った**。

直近10営業日で、空売り勢が実際に「勝った」のは**4/16のみ**だ。本日は逆 — **空売りは参加したが、結果は弱かった**。「空売りフローが存在する」と「空売りが優位」は同義ではない。

### 4.3 ネット空売り残高は依然高水準だが、一貫した再積み増しにはなっていない

```
4/8   1,579,992
4/9   1,598,112
4/10  1,643,197
4/13  1,738,365
4/14  1,748,500
4/15  1,775,454
4/16  1,912,458
4/17  1,876,700
```

残高は4/8から4/16にかけて積み上がった。しかし**4/17に35,758株減少している**：

```
1,912,458 − 1,876,700 = 35,758株の減少
```

水準は依然高い。しかし、**「構造的な空売り再積み増しが確認された」**と宣言するにはデータが不十分だ。現在は**高残高の拮抗ゾーン** — 下降トレンドの解消が確認されたわけでも、空売り主導の支配が確立されたわけでもない。

## 5. 総合的な読み

### 一言で言えば

**本日は空売りが相場の方向性を決めた日ではなかった。外国人・機関の買いと安値からの回復が、空売りの有効性を消耗させた日だった。**

### 根拠

1. 本日の空売り数量（**64,362株**）は直近ピークと比べて低水準。
2. 終値**KRW 54,400**が空売り平均**KRW 54,245**を上回り、新規空売りポジションは含み損。
3. フロー面のサポート：外国人**+28,741株**、機関**+7,960株**。
4. 日中安値**KRW 53,300**から終値**KRW 54,400**への回復。

### 現在はどういう局面か

> **上値はまだ開いていないが、下値の支配力は失われつつある — レンジ再定義局面。**

「強いロング優位」と判断するのは時期尚早。「空売り優位」という読みは誤りだ。**中立、わずかにポジティブ寄り**が正直な評価だ。

## 7. 最後に

Pearl Abyssには依然として空売り残高が積み上がっている。それは変わらない事実だ。しかし直近のデータが示すのは**空売りの有効性の低下**であり、空売り圧力の強化ではない。本日に限って言えば：**空売り数量の減少＋終値が空売り平均を上回る＋外国人・機関の同時買い＋安値からの回復** — 4つが揃った。

最も端的なまとめ：

> **空売りは依然この銘柄にいる。しかし価格支配力は失われつつある。本日は空売りの優位ではなく、空売り圧力に対する買いの吸収だった。**
