## 핵심 요약

* **현재 상황**: 삼성전자 노조가 5/21\~6/7 총파업 예고. 핵심 요구는 DS(반도체) 영업이익의 15%를 성과급으로 고정 지급 + 연봉 50% 상한 폐지. 5/16 중앙노동위원회 추가 조정, 법원 가처분, 정부 긴급조정권이 방어막.
* **파업 손실 추정**: JP모건 21\~35조원, 연합뉴스 인용 업계 추정 최대 100조원. 범위가 넓은 이유는 참여율·라인 차질·재가동 시간이 불확실하기 때문.
* **가격 상쇄 논리**: 2분기 DRAM +58\~63%, NAND +70\~75%면 생산 차질 1\~2% 수준은 수학적으로 상쇄 가능. JP모건도 이 논리로 목표가 35만원 유지.
* **그러나 완전 상쇄는 아니다**: (1) 성과급 제도화에 따른 구조적 인건비 증가는 가격으로 못 덮음. (2) 고객이 SK하이닉스·마이크론으로 주문을 옮기면 가격 상승의 수혜가 삼성에 온전히 귀속 안 됨.
* **본질**: '파업이냐 아니냐'보다 **'초과이익 배분 공식이 어떻게 바뀌느냐'가 중장기 밸류에이션에 더 중요**.

## 1. 지금 무슨 일이 벌어지고 있는가

### 1.1 파업의 규모

삼성전자 노조가 5월 21일부터 6월 7일까지 **18일간 총파업**을 예고했다. 참여 예상 인원은 최대 **5만 명**. 삼성전자 창사 이래 최대 규모의 파업 위기.

5월 11\~12일 중앙노동위원회(중노위) 조정이 진행됐지만 13일 새벽 결렬. 노조는 "구체적인 제도화 없이는 대화할 이유가 없다"는 입장.

### 1.2 노조가 원하는 것 — 단순 임금 인상이 아니다

```
노조 핵심 요구:

1. DS(반도체) 부문 영업이익의 15%를 성과급으로 '고정' 지급
   → 현재: 회사 재량으로 성과급 결정 (EVA 기반 OPI)
   → 요구: 영업이익의 15%를 '공식'으로 제도화

2. 연봉 50% 상한 폐지
   → 현재: 성과급이 연봉의 50%를 넘지 못함
   → 요구: 상한 없앰. 이익이 크면 성과급도 비례해서 커져야

3. OPI 주식보상 제도 확대

왜 이게 중요한가:
→ 삼성전자 1분기 DS 영업이익 53.7조원
→ 15%면 약 8.1조원이 성과급 재원
→ 10%로 해도 5.4조원
→ 기존 OPI 구조와 차이가 조 단위
```

### 1.3 왜 이 이슈가 이렇게 커졌나

**첫째, SK하이닉스와의 보상 격차.** SK하이닉스는 HBM 성공으로 직원 보상이 크게 늘었다. 같은 반도체 업종에서 삼성전자 직원들이 상대적 박탈감을 느끼고 있다.

**둘째, 이익 규모가 너무 커졌다.** 1분기 DS 영업이익만 53.7조원, 이익률 65.7%. 이 정도 초과이익이 발생하면 '누가 가져가느냐'의 문제가 자연스럽게 떠오른다.

**셋째, 반도체 공정의 특수성.** 반도체 공장은 24시간 연속 공정이다. 중간에 멈추면 웨이퍼가 폐기되고, 수율이 떨어지고, 재가동에 시간이 걸린다. 그래서 파업 위협만으로도 공장 운영에 영향을 준다. 실제로 삼성전자는 이미 신규 웨이퍼 투입을 줄이는 '웜다운(warm-down)'에 들어간 것으로 보도됐다.

## 2. 파업 손실은 얼마인가 — 범위가 넓은 이유

### 2.1 추정치들

| 출처 | 손실 추정 | 성격 |
| --- | ---: | --- |
| JP모건 | 21\~35조원 | 노무비+생산차질 포함 |
| 매출 기회손실 (JP모건) | 약 4.5조원 | 직접 매출 손실만 |
| 업계 추산 (연합뉴스) | 직접 40조, 간접 포함 100조 | 최악 시나리오 |

### 2.2 왜 이렇게 범위가 넓은가

```
불확실한 변수:
1. 실제 참여율 — 5만 명 중 몇 명이 진짜 파업하는가
2. 핵심 라인 인력 — DS(반도체) 공정 엔지니어가 참여하는가
3. 필수 인력 — 법원 가처분으로 안전·필수 인력이 빠지는가
4. 파업 기간 — 18일 전체인가, 며칠 만에 타결되는가
5. 재가동 시간 — 멈춘 라인을 다시 돌리는 데 얼마나 걸리는가

간단 검산:
연합뉴스가 언급한 '하루 약 2.2조원 손실'
= 직접 손실 40조 ÷ 18일 = 2.2조원/일

2018년 평택 정전 사례: 28분간 약 500억원
→ 하루 환산 시 약 2.6조원
두 숫자가 비슷한 오더 → 검증에 참고 가능
다만 과거 정전과 현재 파업은 상황이 다르므로 단순 적용은 부정확
```

## 3. 가격 상승이 파업 손실을 상쇄할 수 있는가 — JP모건의 논리

### 3.1 JP모건이 뭐라고 했나

JP모건(Jay Kwon 팀)은 5월 13일 업데이트에서 **투자의견 Overweight, 목표가 35만원 유지**. 핵심 논리: "메모리 가격 상승이 예상보다 강해서 파업 손실을 상당 부분 흡수할 수 있다."

| 항목 | JP모건 수치 |
| --- | ---: |
| 2분기 DRAM 계약가 상승 | +58\~63% |
| 2분기 NAND 계약가 상승 | +70\~75% |
| 기존 예상치 | +40\~50% |
| 생산 차질 영향 (DS 매출 기준) | 1\~2% |

### 3.2 이 논리가 수학적으로 맞는가

```
메모리 매출 = 출하량 × 평균 판매 단가(ASP)

파업이 출하량을 줄여도,
ASP 상승률이 출하량 감소율보다 크면 매출은 방어된다.

상쇄 조건:
(1 + ASP 상승률) × (1 - 출하량 감소율) ≥ 1

생산 차질이 1\~2%라면:
→ 매출 방어에 필요한 ASP 상승률 = 약 1\~2%
→ 실제 가격 상승: DRAM +58\~63%, NAND +70\~75%
→ 필요량(1\~2%)보다 수십 배 큰 가격 상승

∴ 직접 생산 차질만 놓고 보면 수학적으로 상쇄 가능
```

더 중요한 계산 — **기존 예상 대비 '초과' 가격 상승분**:

```
기존 예상: DRAM +45%, NAND +45% (중간값)
신규 전망: DRAM +60.5%, NAND +72.5% (중간값)

초과 효과:
DRAM: 1.605 / 1.45 - 1 = +10.7%
NAND: 1.725 / 1.45 - 1 = +19.0%

→ 기존 모델보다 DRAM ASP가 약 11%, NAND ASP가 약 19% 더 강해짐
→ 이 초과분이 파업 손실을 상쇄하는 논리
```

### 3.3 맞는 부분과 부족한 부분

**맞는 부분**: 직접 생산 차질(DS 매출 1\~2%)은 가격 상승으로 충분히 덮인다. 2분기 실적 추정치는 오히려 상향될 수 있다. **단기 실적에 한정하면 JP모건 논리는 타당.**

**부족한 부분 세 가지:**

```
1. 노무비 구조 변화는 가격으로 못 덮는다
   노조가 요구하는 '영업이익 15% 고정 성과급'이 제도화되면,
   이건 매 분기 영업이익에서 15%가 인건비로 빠져나가는 '구조적 비용'이 된다.
   가격이 올라도 이 비용은 같이 올라간다.

   1분기 기준 15% = 8.1조원
   → 이건 가격 상승으로 '상쇄'하는 게 아니라 '분배'가 바뀌는 것

2. 고객이 주문을 옮기면 가격 상승의 수혜가 안 온다
   HBM, 고용량 서버 DRAM, 엔터프라이즈 SSD는
   고객이 한 번 공급선을 바꾸면 되돌리기 어렵다.

   삼성전자 파업 → 납기 불안 → 고객이 SK하이닉스/마이크론에 주문
   → 메모리 가격은 오르지만 수혜는 경쟁사가 가져감

   TrendForce도 "직접 손실보다 주문 이동(order shift)이 더 큰 위험"이라고 지적

3. 메모리 업사이클은 언젠가 끝난다
   성과급이 '고정 비율'로 제도화되면,
   업사이클 때는 문제없지만 다운사이클 때 비용 경직성이 커짐
   → 이건 삼성전자의 장기 ROIC(투하자본수익률)에 영향
```

## 5. 투자자가 진짜 봐야 할 것 — '파업'이 아니라 '배분 공식'

### 5.1 세 가지 시나리오와 주가 영향

| 시나리오 | 내용 | 주가 영향 |
| --- | --- | --- |
| **A. 합의·유예** | 중노위 조정이나 정부 개입으로 파업 회피. 성과급은 기존 OPI + 특별보상 | **단기 반등**. 리스크 프리미엄 제거, 메모리 업황 재부각 |
| **B. 부분 파업** | 일부 인력 파업, 핵심 라인 유지. 참여율 낮음 | **중립\~소폭 부정**. 노이즈 지속, 실적 영향 제한적 |
| **C. 대규모 파업** | DS 핵심 인력 참여, 웜다운 확대, 고객 allocation 이동 | **의미 있는 하락**. 이익 추정치 하향 + 멀티플 디레이팅 |

### 5.2 시나리오보다 중요한 것 — 성과급 합의의 형태

```
파업이 끝나더라도 남는 질문:
"성과급을 어떻게 제도화했는가?"

가장 좋은 경우 (주주 관점):
→ 기존 OPI 구조 유지 + 업황에 따른 변동형 특별보상
→ 초과이익이 클 때 직원에게도 일부 배분하되, 고정 비율은 아님
→ 다운사이클 때 비용 경직성이 없음

가장 나쁜 경우 (주주 관점):
→ DS 영업이익의 15%를 고정 성과급으로 제도화
→ 매 분기 영업이익에서 15%가 인건비로 빠져나감
→ 업사이클 때는 괜찮지만, 다운사이클 때 이익 방어가 어려워짐
→ 장기 ROIC(투하자본수익률)가 구조적으로 낮아짐
```

**이 '배분 공식'이 어떻게 결정되느냐가 삼성전자의 중장기 밸류에이션에 더 중요하다.** 파업이 하루 만에 끝나도, 성과급이 15% 고정으로 제도화되면 주주 귀속 이익은 구조적으로 줄어든다.

## 6. 삼성전자 주가는 지금 어디에 있는가

### 6.1 실적은 매우 강하다

이전 글에서 분석한 대로:
* 1분기 매출 133.9조원, 영업이익 57.2조원
* DS 영업이익 53.7조원, 이익률 65.7%
* 연환산 영업이익 약 228.8조원 → 현재가 기준 PER 약 9.6배

### 6.2 증권사 전망

## 9. 마지막 한 줄

삼성전자 파업의 본질은 '공장이 멈추느냐'가 아니다. <strong>'메모리 슈퍼사이클의 초과이익을 노동자와 주주 중 누가 얼마나 가져가느냐'</strong>가 본질이다.

단기적으로 JP모건의 논리는 맞다 — DRAM 가격 +60%, NAND +73%면 생산 차질 1\~2%는 수학적으로 상쇄된다. 하지만 성과급이 영업이익의 15%로 고정 제도화되면, 이건 가격 상승으로 덮을 수 있는 일시적 비용이 아니라 **이익 구조가 바뀌는 것**이다. 그리고 파업으로 고객이 HBM·서버 DRAM 주문을 SK하이닉스·마이크론으로 옮기면, 가격 상승의 수혜가 삼성전자에 온전히 오지 않는다.

전면 셧다운까지 갈 가능성은 낮다 — 중노위, 법원, 정부 모두 방어막이 있다. 하지만 '파업이 안 일어난다'와 '리스크가 없다'는 다른 말이다. 보유자는 과잉 반응할 필요 없지만, 신규 매수자는 5/16 조정 결과와 5/21 파업 여부를 확인한 뒤 들어가는 것이 합리적이다.

**지금 필요한 것은 용기가 아니라 이벤트 리스크 관리다.**
