엔비디아의 Vera Rubin NVL72 한 대에는 Vera CPU 36개와 LPDDR5X 메모리 54TB가 들어간다. CPU 한 개당 1.5TB다. 그런데 6월 TrendForce는 엔비디아가 차세대 Vera용 SoCAMM 모듈의 용량을 절반으로 낮추고 전체 모듈 출하량과 Vera CPU 생산량은 늘리기로 했다고 전했다.

겉으로 보면 메모리 용량 축소는 부품사에 악재다. 기판 수요를 매출 비트 수로만 보면 그렇다. 그러나 심텍이 만드는 것은 DRAM 칩이 아니라 SoCAMM 모듈용 PCB다. 심텍의 물량은 메모리 총용량보다 생산되는 CPU 수와 모듈 수에 더 민감하다. LPDDR 부족 때문에 CPU 한 대당 탑재 용량을 낮추더라도 더 많은 Vera CPU와 모듈을 출하한다면 기판 수요는 줄지 않을 수 있다.

공식 자료, 산업조사, 증권사 추정, 실제 주가와 수급을 나눠 보면 사업 논리는 살아 있지만 주가의 바닥은 아직 확인되지 않았다.

## TL;DR

* NVIDIA 공식 사양에서 Vera Rubin NVL72는 Vera CPU 36개와 LPDDR5X 54TB를 쓴다. 별도 Vera CPU 랙은 1U 트레이에 최대 4개 노드, 한 랙에 최대 256개 CPU까지 확장된다. Vera가 GPU 랙의 보조 CPU에서 독립 서버 플랫폼으로 넓어지는 경로가 확인됐다.
* TrendForce가 전한 SoCAMM 용량 축소는 수요 둔화가 아니라 LPDDR 공급 부족에 대한 대응이다. 엔비디아는 모듈당 용량을 낮추는 대신 전체 모듈 출하량과 Vera CPU 생산량을 늘리려 한다.
* 심텍은 공식 제품 목록에 SoCAMM PCB를 올려두고 있다. 다만 메모리 3사 동시 공급과 높은 점유율은 회사 공시가 아니라 증권사 추정이다. 실제 고객별 매출과 점유율은 공개되지 않았다.
* 7월 16일 심텍 종가는 10만7,400원으로 하루 11.68% 하락했다. 20거래일 동안 기관은 2,362억원, 투신·연기금·보험을 묶은 real money는 1,864억원을 순매수했지만 외국인은 1,611억원, 프로그램은 1,859억원을 순매도했다. 중기 기관 매수와 단기 매도 압력이 충돌한다.
* 2026년 예상 PER 27.1배는 싸지 않다. 2027년 예상 EPS 6,655원을 적용하면 16.1배로 낮아지지만 SoCAMM 매출과 이익률이 계획대로 커져야 한다.
* 판단은 <strong>Conditional Buy</strong>다. 10만4천원 안팎의 지지와 외국인·프로그램 매도 둔화를 먼저 확인하는 경로, 또는 11만3,800원 부근을 거래량과 함께 회복하는 경로가 더 안전하다.

## 1. 먼저 버릴 숫자와 남길 숫자

첨부 분석에 등장한 2030년 서버 CPU 시장 1,700억달러, CPU와 GPU의 1대1 비율, 2027년 DRAM 부족은 이미 여러 증권사와 기업 설명에 반영된 큰 흐름이다. 산업 방향을 이해하는 데는 유용하지만 심텍만의 초과수익 근거는 아니다.

심텍을 볼 때 더 중요한 숫자는 다음 네 가지다.

| 항목 | 왜 중요한가 |
|---|---|
| 출하되는 Vera CPU 수 | SoCAMM 플랫폼의 설치 기반을 결정한다 |
| CPU 한 개당 SoCAMM 모듈 수 | 심텍 모듈 PCB의 직접 물량 변수다 |
| 심텍의 고객별 공급 점유율 | 산업 성장 중 심텍에 귀속되는 몫을 결정한다 |
| 모듈 PCB 단가와 원가 | 매출 증가가 영업이익으로 바뀌는 비율을 결정한다 |

Vera CPU 500만~600만 대, CPU당 모듈 8개라는 가정을 쓰면 연간 모듈은 4,000만~4,800만 개가 된다. 중간값 550만 대를 넣으면 4,400만 개다. 그러나 이 계산은 NVIDIA가 공식 발표한 출하계획이 아니다. 증권사와 업계의 CPU 출하 추정, 공개된 시스템 구성에서 나온 시나리오다.

따라서 4,400만 개를 매출 추정치로 바로 쓰면 안 된다. 물량의 크기를 가늠하는 예시일 뿐이다.

## 2. NVIDIA 공식 자료가 확인한 Vera의 확장 경로

### 2.1 NVL72 안에서 Vera는 36개가 들어간다

[NVIDIA Vera Rubin NVL72 공식 페이지](https://www.nvidia.com/en-us/data-center/vera-rubin-nvl72/)가 제시하는 구성은 다음과 같다.

| 항목 | 공식 사양 |
|---|---:|
| Rubin GPU | 72개 |
| Vera CPU | 36개 |
| CPU 메모리 | LPDDR5X 54TB |
| Vera CPU 한 개당 메모리 | 1.5TB |
| Vera CPU 코어 | 88개 |

GPU 한 개당 CPU 한 개라는 단순한 구조는 아니다. NVL72에서는 GPU 2개당 Vera CPU 1개다. 그럼에도 72개 GPU 랙 한 대에 CPU가 36개씩 필요하다는 점은 CPU와 LPDDR 계층이 AI 시스템 원가의 큰 항목으로 올라왔음을 보여준다.

### 2.2 Vera는 GPU 랙 밖에서도 팔린다

더 중요한 변화는 독립 Vera CPU 시스템이다. [NVIDIA 기술 블로그](https://developer.nvidia.com/blog/?p=114004)는 Vera CPU 랙이 1U 트레이에 최대 4개 노드를 담고, 한 랙에서 최대 256개 Vera CPU까지 확장될 수 있다고 설명한다. 단일 소켓과 듀얼 소켓 시스템은 소켓당 최대 1.5TB LPDDR5X를 지원한다.

NVIDIA는 Cisco, Dell, HPE, Lenovo, Supermicro가 2026년 하반기 Vera 기반 시스템을 내놓을 예정이라고 밝혔다. 일정이 지켜진다면 Vera의 수요처는 NVL72 조립 대수에만 묶이지 않는다.

```text
Vera 수요처

Vera Rubin NVL72의 CPU 계층
+ 독립 Vera CPU 서버
+ 추론·에이전트용 CPU 랙
+ 주요 OEM의 단일·듀얼 소켓 시스템
```

심텍 투자 논리가 GPU 랙 생산량 하나에만 의존하지 않는 이유다. 독립 CPU 시스템이 늘면 SoCAMM 모듈과 PCB의 수요 기반도 넓어진다.

## 3. SoCAMM 용량 축소는 무엇을 뜻하나

[TrendForce의 2026년 6월 10일 분석](https://www.trendforce.com/presscenter/news/20260610-13090.html)은 2027년 엔비디아가 필요로 하는 LPDRAM의 약 60%만 예비 배분으로 충족될 수 있다고 봤다. 공급 부족에 대응해 엔비디아는 다음과 같은 조정을 선택한 것으로 전해졌다.

1. Vera용 SoCAMM 모듈 한 개의 메모리 용량을 낮춘다.
2. 전체 모듈 출하량을 늘린다.
3. 같은 LPDRAM 공급량으로 더 많은 Vera CPU를 만든다.
4. 독립 Vera CPU 랙의 시장 침투를 높인다.

이 구조에서는 두 수요가 다른 방향으로 움직일 수 있다.

| 수요 단위 | 용량 축소의 영향 |
|---|---|
| DRAM 비트·GB | CPU 한 대당 탑재량 감소 가능 |
| SoCAMM 모듈 수 | CPU 출하 확대가 더 크면 증가 가능 |
| 모듈 PCB | 모듈 수에 비례하므로 증가 가능 |
| 소켓·커넥터·모듈 테스트 | 모듈 수가 늘면 함께 증가 가능 |

이를 심텍 기준으로 쓰면 다음 식이 된다.

```text
심텍 SoCAMM PCB 물량
= Vera CPU 출하량
× CPU당 SoCAMM 모듈 수
× 모듈당 PCB 수
× 심텍 공급 점유율
```

반대로 메모리업체의 SoCAMM 매출은 대체로 CPU 수 × CPU당 탑재 GB × GB당 가격에 가깝다. 같은 뉴스가 DRAM 업체와 모듈 PCB 업체에 다른 영향을 줄 수 있다.

### 공식 사양과 TrendForce 보도가 아직 맞지 않는 부분

NVIDIA의 현재 공식 페이지는 Vera CPU당 최대 1.5TB LPDDR5X를 제시한다. TrendForce는 차세대 모듈의 용량 축소를 전했다. 두 자료가 동시에 존재하므로 다음 중 어느 경우인지 아직 확정할 수 없다.

* NVIDIA 페이지가 최대 구성 또는 기존 설계를 계속 표시하는 경우
* 용량 축소가 특정 고객·제품 구성에만 적용되는 경우
* 최종 양산 사양이 아직 확정되지 않은 경우

이 불일치는 감춰야 할 잡음이 아니라 투자 판단의 확인 항목이다. 실제 양산 시스템의 모듈당 용량, CPU당 모듈 수, 고객별 구성은 출시 제품과 회사 공시로 확인해야 한다.

## 4. 심텍은 SoCAMM2에서 무엇을 공급하나

### 4.1 회사가 공식 확인한 부분

[심텍 공식 제품 페이지](https://www.simmtech.com/product/board06.aspx)는 SoCAMM PCB를 AI용 차세대 메모리 모듈 제품으로 분류한다. 회사가 제시한 주요 사양은 다음과 같다.

| 항목 | 심텍 공식 설명 |
|---|---|
| 기반 메모리 | LPDDR |
| 용도 | AI용 차세대 메모리 모듈 |
| 층수 | 10~12층 |
| 핵심 요구 | 낮은 유전율·손실, 낮은 열팽창계수, 임피던스 제어 |
| 표면처리 | 선택적 ENIG |

SoCAMM PCB가 심텍의 제품 목록에 있다는 점은 확인됐다. 회사는 일반 메모리 모듈 PCB와 메모리 패키지 기판도 생산한다. AI 서버의 LPDDR 수요가 모듈과 패키지 양쪽으로 커질 때 대응할 제품 기반은 있다.

### 4.2 증권사 추정으로만 확인되는 부분

심텍이 삼성전자, SK하이닉스, 마이크론의 SoCAMM2 생산에 모두 참여하고 점유율이 높다는 주장은 회사 공시가 아니다. [심텍 홈페이지에 게시된 대신증권 리포트](https://www.simmtech.com/upload/media/file/639179687810021917.pdf)의 분석과 시장 추정이다.

따라서 다음 표현을 구분해야 한다.

| 표현 | 증거 수준 |
|---|---|
| 심텍은 SoCAMM PCB 제품을 보유한다 | 회사 공식 확인 |
| 심텍은 메모리 3사 SoCAMM2 생산에 참여한다 | 증권사 추정 |
| 심텍의 공급 점유율이 1위다 | 증권사 추정 |
| 고객별 매출과 실제 점유율 | 공개되지 않음 |
| SoCAMM이 심텍 이익에 기여하는 정확한 시점 | 공개되지 않음 |

이 구분이 필요한 이유는 고객 인증과 양산 물량이 기판 사업의 경제성을 좌우하기 때문이다. 제품을 만들 수 있다는 사실과 높은 점유율로 반복 매출을 올린다는 사실 사이에는 고객 인증, 수율, 단가, 가동률이라는 단계가 남아 있다.

### 4.3 SK하이닉스의 양산은 플랫폼의 실체를 확인한다

[SK하이닉스 공식 발표](https://news.skhynix.com/mass-production-socamm2-192gb/)에 따르면 회사는 2026년 4월 1cnm LPDDR5X를 사용한 192GB SoCAMM2의 양산을 시작했다. 제품은 NVIDIA Vera Rubin 플랫폼을 대상으로 한다.

이는 SoCAMM2가 장기 연구과제가 아니라 실제 양산 단계에 들어왔음을 보여준다. 다만 SK하이닉스의 모듈 양산이 심텍의 고객별 공급량을 자동으로 증명하지는 않는다. 심텍이 2분기와 3분기 실적에서 SoCAMM 매출, 고다층 제품 비중, 가동률 개선을 얼마나 보여주는지가 별도로 필요하다.

## 5. 실적은 회복 중이지만 아직 기대를 증명하는 단계다

심텍은 2025년 연간 매출 1조4,106억원, 영업이익 119억원에 그쳤고 순손실 1,642억원을 기록했다. 2026년 1분기 연결 매출은 약 4,224억원, 영업이익은 약 137억원으로 회복했다. 2분기 컨센서스는 회복 속도가 더 빨라질 것으로 본다.

| 항목 | 1Q26 실제 | 2Q26 컨센서스 | 2Q 순차 변화 |
|---|---:|---:|---:|
| 매출 | 약 4,224억원 | 4,669억원 | 약 +10.5% |
| 영업이익 | 약 137억원 | 473억원 | 약 +245% |
| 영업이익률 | 약 3.2% | 약 10.1% | +6.9%p |

1분기 실제치는 [심텍 분기보고서](https://kind.krx.co.kr/external/2026/05/14/000890/20260514001959/11013.htm), 2분기 예상치는 Research OS 로컬 DB가 7월 16일 수집한 Naver·WiseReport 컨센서스다.

## 8. P×Q×C로 다시 보는 심텍

### P: 고다층 SoCAMM PCB의 단가

SoCAMM PCB는 10~12층, 낮은 유전손실, 낮은 열팽창, 정밀한 임피던스 제어가 필요하다. 일반 메모리 모듈보다 제조 난도가 높아 제품 구성 개선에 도움이 될 수 있다. 그러나 고객별 단가와 판가 인상 구조는 공개되지 않았다.

### Q: Vera CPU와 SoCAMM 모듈 수

Q가 가장 큰 상방 변수다. NVL72의 양산뿐 아니라 독립 Vera CPU 랙과 주요 OEM 서버가 2026년 하반기에 예정돼 있다. TrendForce 보도대로 엔비디아가 모듈당 용량을 낮추고 모듈 출하를 늘리면 심텍의 Q는 DRAM 비트보다 빠르게 증가할 수 있다.

### C: 원재료, 금도금, 수율, 가동률

매출 성장이 영업이익으로 이어지는지를 결정하는 항목이다. PCB는 금과 구리 가격, BT 소재, 도금 비용, 초기 수율에 민감하다. 2분기 영업이익률 컨센서스 10.1%를 달성하려면 고부가 제품 증가와 원가 전가가 함께 확인돼야 한다.

## 10. 투자 판단과 확인 순서

### 판단: Conditional Buy

사업 논리는 이전보다 선명해졌다. NVIDIA 공식 자료는 Vera의 독립 서버 확장을 확인했고, TrendForce 보도는 용량 축소가 수요 감소가 아닌 공급 부족 대응임을 보여준다. 심텍은 공식적으로 SoCAMM PCB 제품을 보유한다.

그러나 주가는 아직 외국인·프로그램 매도에 눌려 있고 2026년 이익 기준으로 싸지 않다. 따라서 두 가지 진입 경로를 나눠 보는 편이 낫다.

| 경로 | 확인 조건 |
|---|---|
| 지지 확인 | 10만3,700원~10만6,000원에서 저점을 지키고 외국인·프로그램 매도가 둔화되는지 확인 |
| 추세 확인 | 11만3,800원 안팎을 거래량과 함께 회복하고 기관·외국인 중 한 주체가 순매수로 돌아서는지 확인 |

12만1,600원~12만4,000원을 회복하면 단기 추세 훼손을 상당 부분 되돌린다. 반대로 9만9,600원을 종가 기준으로 이탈하고 회복하지 못하면 가격상 가설을 다시 점검해야 한다.

### 촉매

* 2026년 2분기 실적에서 매출 4,669억원, 영업이익 473억원의 충족 여부
* SoCAMM 매출, 고다층 기판 비중, 가동률에 대한 회사 설명
* 2026년 하반기 Vera 기반 OEM 서버의 실제 출시
* NVIDIA의 최종 양산 사양과 독립 Vera CPU 랙 출하 확대
* SK하이닉스·삼성전자·마이크론의 SoCAMM2 양산 및 고객 인증 업데이트

### 무효화 조건

* SoCAMM 초도 물량 뒤에 반복 주문이 붙지 않는 경우
* 모듈당 용량 축소가 총 모듈 출하 증가로 이어지지 않는 경우
* 심텍의 고객별 점유율이 증권사 추정보다 낮은 것으로 확인되는 경우
* 2분기 영업이익이 컨센서스를 의미 있게 밑돌고 원재료·수율 부담이 반복되는 경우
* Vera Rubin 또는 독립 Vera CPU 시스템의 출시가 장기간 늦어지는 경우
* 9만9,600원 이탈과 외국인·프로그램 매도 확대가 함께 나타나는 경우

## 11. 최종 결론

Vera CPU 공급 확대는 심텍의 SoCAMM2 투자 논리를 강화한다. 특히 LPDDR 공급 부족 때문에 모듈 용량을 낮추면서 전체 모듈과 CPU 출하를 늘리는 경로는 기판업체에 불리하다고 단정하기 어렵다. DRAM 업체는 비트와 GB를 팔고, 심텍은 모듈마다 들어가는 PCB를 판다. 서로 다른 단위다.

다만 아직 확인되지 않은 항목도 많다. Vera 출하량, CPU당 최종 모듈 수, 심텍의 실제 고객별 점유율, SoCAMM 매출, 고다층 기판의 이익률은 공개자료만으로 확정할 수 없다. 4,400만 모듈 같은 계산은 유용한 시나리오지만 공식 가이던스가 아니다.

현재 주가는 고점 대비 35% 가까이 조정됐지만 외국인과 프로그램 매도가 끝났다는 증거도 없다. 그러므로 이 아이디어는 “많이 빠진 성장주”보다 “실적과 수급을 함께 확인해야 하는 조건부 매수 후보”로 보는 편이 맞다. 2분기 영업이익과 SoCAMM 매출 설명이 첫 번째 검증이고, 2026년 하반기 Vera 독립 서버 출하가 두 번째 검증이다.

## 근거와 한계

### 확인된 사실

* NVIDIA 공식 사양은 NVL72에 Vera CPU 36개와 LPDDR5X 54TB를 제시한다.
* NVIDIA는 한 랙에 최대 256개 Vera CPU를 담는 독립 CPU 시스템과 2026년 하반기 OEM 출시 계획을 공개했다.
* TrendForce는 LPDRAM 부족으로 모듈당 용량을 낮추되 전체 모듈 출하와 Vera CPU 생산량을 늘리는 조정을 전했다.
* SK하이닉스는 Vera Rubin용 192GB SoCAMM2 양산을 발표했다.
* 심텍은 공식 제품 목록에 SoCAMM PCB를 두고 있다.
* 7월 16일 심텍 종가, 수급, 컨센서스는 로컬 DB에서 재검증했다.

### 추정

* Vera CPU 500만~600만 대와 CPU당 모듈 8개를 사용한 연 4,000만~4,800만 모듈 계산
* 심텍이 메모리 3사에 동시에 공급하고 높은 점유율을 보유한다는 주장
* 용량 축소 이후 모듈 수 증가가 심텍의 매출과 이익에 미치는 크기

### 공개자료로 확인되지 않은 항목

* NVIDIA의 2027년 Vera CPU 공식 출하 목표
* 최종 양산 시스템의 CPU당 SoCAMM 모듈 수와 모듈당 용량
* 심텍의 고객별 SoCAMM 매출, 점유율, 단가, 이익률
* 2분기 실적 발표일과 회사의 SoCAMM 매출 가이던스
* TrendForce가 전한 용량 축소와 NVIDIA 공식 1.5TB 사양의 정확한 적용 범위

공개자료와 로컬 시장 데이터를 바탕으로 한 정보성 분석이며, 개인의 투자기간·가격·위험한도를 반영한 투자자문이 아니다.
