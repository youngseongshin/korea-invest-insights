## 补丁内容——八大类

### 1. 难度系统（最重磅改动）

可在游戏途中自由切换简单 / 普通 / 困难。简单模式降低伤害、放宽弹反/闪避窗口、减少Boss反击频率；困难模式新增Boss招式、收紧无敌帧。**Boss再挑战**已确认为**下一个版本**内容——内容管线的可见度已延伸至下一个补丁。

这是本次更新中最关键的改动。难度模式**拓宽了目标受众（TAM）**：让以剧情为主的休闲玩家能够完成此前因难度而卡关的内容，同时不影响核心硬核玩家体验。这正是《巫师3》上线后采用的同一机制。

### 2. 住宅/仓储大改

- **坚固采集物收纳箱** — 1,000格
- **收藏品收纳箱** — 1,000格
- **衣橱** — 最多1,000格
- Kuku冷却箱 / 强化版 — 40 / 330格（食材专用）
- 房型选择 — 4种布局（紧凑型 / 标准型 / 宽敞型 / 帕伊鲁内斯风格）
- **核心体验优化**：仓库内物品可直接用于制作/烹饪，无需移至主背包

这正是"玩家差评的真正根源"——已在一个补丁内得到解决。

### 3. 宠物系统扩展
新增5只猫咪、全新宠物、深渊黑狼宠物化，以及**宠物饰品栏位**（羁绊印记）、重命名功能。

### 4. 新增内容
四个巡逻司法区域、三款新武器（星光之剑、树枝、坚固树枝）、巴尔泰昂套装、波罗林秘密商店（宠物装备）、13款新纹身。

### 5. 画面优化
远景改善，部分2D资源替换。

### 6. 操控优化
键鼠及手柄预设（经典模式），闪避/翻滚输入互换，瞄准控制优化。

### 7. 战斗平衡
**移除**Boss重攻击时的无敌帧；状态异常的属性伤害提升；新增技能：力掌脉冲（三段蓄力）、投掷武器（双持）。

### 8. 品质优化（QoL）
道具锁定（防误售/误丢弃）、休息后立即起身、深渊神器可通过技能观察重新获取。

## 修正后的投资论——估值重新定价

解读本次补丁的正确视角是：**它消除了空头论点中的最大支撑点**（QoL/难度/好评率卡在83%），同时**打开了内容管线的能见度**（Boss再挑战已确认为下一版本内容）。

采用**P/E 15×**（适用于已证明BDO式长效运营能力的作品，而非适用于一次性买断作品的12×）：

| 项目 | 补丁前（4/22） | **补丁后（4/23）** |
|---|---|---|
| 第5周衰退率 | -15至-20% | **-15%（已确认）** |
| 第6周衰退率 | -15% | **-10%（补丁催化）** |
| D53（5月12日）累计销量 | 600-610万 | **610-625万** |
| 2026年全年销量 | 1,000-1,100万 | **1,050-1,200万** |
| CD 2026年预计营收 | ₩5,000-5,300亿 | **₩5,200-5,600亿** |
| 合并2026年预计营收 | ₩9,700亿-1.0兆 | **₩9,900亿-1.03兆** |
| EPS | ₩5,400-5,500 | **₩5,500-5,750** |
| **合理价值（P/E 15×）** | — | **约₩82,500-86,250** |
| 对比现价₩53,100 | — | **+55至+62%** |

12×→15×的估值倍数扩张本身就是一个论点押注。论据：Pearl Abyss已不再是一家单纯依赖《赤色沙漠》的公司，而是一家**拥有BDO历史资产和CD持续内容特许经营权的多IP长效运营公司**。稳定盈利的多IP长效运营型韩国游戏公司的可比估值为15-20×。15×是该区间的**低端**。

保守的P/E 12×基础情景下，合理价值约₩66,000-69,000（+24-30%），这是价值底线。15×是市场认可长效运营能力后的结果。
