## 核心摘要

从指数层面看，2026年上半年似乎是一段普遍的风险偏好上升期。但深入内部，市场远比表面更为集中。韩国是KOSPI超大盘股抱团市场。美国则是AI基础设施市场，领涨方向从超大盘AI平台转向存储器、存储设备、服务器、半导体设备与电力板块。两个市场都不是"全面上涨"行情。

韩国指数的表现堪称惊人。KOSPI从2025年12月30日的4,214.17点涨至2026年6月30日的8,476.48点，涨幅达101.1%。KOSDAQ则从925.47跌至916.18，下跌1.0%。在约2,700只韩国股票中，856只上涨，1,788只下跌。中位股票收益率为-12.6%。指数翻番，但普通股票处于弱市之中。

美国市场的结构同样比指数所呈现的更为集中。标普500上涨9.2%，纳斯达克上涨11.1%，但真正的主线是AI基础设施瓶颈。SanDisk、Micron、Western Digital、Dell、Marvell、ARM、Applied Materials和Lam Research的表现均远强于NVIDIA。市场在买入的，是AI工厂实际形成瓶颈的环节。

下半年策略不是简单的"寻找低估股票"。更好的原则是在回调时买入有真实营收和合同支撑的已验证瓶颈标的。设备、游戏和生物科技事件型标的（期权型）应缩小仓位。存储器、eSSD、MLCC、基板、电力、造船、国防及大型生物科技平台等持续性营收瓶颈值得更多关注。

## 3. 韩国不是外资净买入市场

| 养老金及其他 | -KRW 7.0T |

关键在于，韩国并非简单的外资净买入市场。外资大量卖出Samsung Electronics和SK Hynix，国内散户、金融投资账户和机构承接了这些卖盘。

| 个股 | 年初至今涨幅 | 外资累计流向 |

外资在核心超大盘股上涨期间持续卖出，这并非正常的外资主导型上涨行情，而是一场国内流动性承接的市场。

外资并非卖出一切，而是轮动至部分瓶颈标的和全球基金易于持仓的资产，如Samsung Electro-Mechanics、Doosan Enerbility、Samsung SDI、FADU、Hanwha Ocean和Sanil Electric。准确的表述不是"外资买入韩国"，而是"外资卖出最大市值存储股，国内流动性承接，外资再配置至部分第二层瓶颈标的"。

## 4. 隐性买方是金融投资，而非纯多头真实资金

机构买入并不等同于纯多头建仓。最强的机构子类别是金融投资，而非投资信托、私募基金或养老金。

| 资金类别 | 解读 |
|---|---|
| 金融投资 | ETF、衍生品、程序化及再平衡相关资金流 |
| 投资信托 | 更可能代表纯多头基金需求，但上半年表现疲弱 |
| 私募基金 | 主题、多空及事件驱动资金混合，累计净流出 |
| 养老金及其他 | 长期真实资金，但上半年净卖出 |
| 散户 | 超大盘AI基础设施资金流与杠杆的关键承接方 |

这一区别至关重要。真实资金能够支撑趋势。金融投资和程序化资金流虽然力量强大，但更容易受到期货到期、ETF规模限制、衍生品持仓和杠杆机制的影响。

上半年韩国市场既是基本面重估市场，也是衍生品/ETF市场。混淆两者会导致反复犯错。

## 5. KOSDAQ缺的是买方，不是故事

KOSDAQ不缺故事：生物科技、机器人、CPO、SOCAMM、半导体设备、游戏与AI软件应有尽有。但指数仍然落后，因为买方基础薄弱。

6月23日的中期快照已经清晰地说明了这一点：

| 指数 | 年初至今 | 距高点 |
|---|---:|---:|
| KOSPI | +90.4% | -10.0% |
| KOSDAQ | -5.7% | -27.3% |

6月30日最终结构相同：KOSPI +101.1%，KOSDAQ -1.0%。

KOSDAQ表现疲弱的原因：

1. 散户资金忙于承接Samsung Electronics、SK Hynix及大型AI基础设施股票。
2. 机构真实资金未能广泛买入KOSDAQ。
3. 外资买入范围有限，集中于FADU、Jeju Semiconductor和Simmtech等少数标的。
4. 利率与汇率压力压低了生物科技和未盈利成长股的估值倍数。

KOSDAQ复苏的条件：

```text
KOSDAQ 复苏 =
散户资金回流
+ 机构真实资金买入
+ Samsung Electronics 与 SK Hynix 横盘整理而非下跌
+ KOSDAQ 盈利预测企稳
```

若Samsung和SK Hynix大幅下跌，资金不会自动轮动至KOSDAQ——风险规避情绪会先行发酵。KOSDAQ最好的环境不是超大盘股崩盘，而是超大盘股横盘整理。

## 7. 宏观并非纯粹顺风

| 资产或指标 | 上半年变动 |
|---|---:|
| 美元指数 | +3.1% |
| 美元/韩元 | +7.9% |
| 美国10年期国债收益率 | 4.16% 至 4.39% |
| WTI原油 | +24.1% |
| 黄金 | -6.8% |
| 比特币 | -33.2% |

这不是一个干净的估值扩张宏观环境。美元走强，韩元走弱，美国国债收益率上升，油价上涨，黄金和比特币下跌。

股票之所以仍能上涨，是因为AI基础设施的盈利周期压倒了宏观压力。这对下半年至关重要。若利率、美元和油价再度同步上涨，即便需求依然强劲，最拥挤的瓶颈股也可能面临估值压缩。

现实风险情境如下：

```text
AI需求保持强劲
+ 盈利持续改善
+ 利率、美元与油价维持高位
+ 估值与杠杆资金流率先出现裂缝
```

这与NVIDIA/Samsung/SK Hynix框架中讨论的股票弹性问题如出一辙。

## 8. 非共识真相一：领涨方向从算力转向存储

第一个非共识点涉及美国AI市场的领涨格局。NVIDIA仍是AI的核心，但边际股票表现已向Micron、SanDisk、Western Digital、Dell、Marvell和ARM转移。

这并不意味着算力不再重要，而是边际回报沿着产业链向下游移动了。

| 问题 | 含义 |
|---|---|
| 该公司是AI资本支出的成本项吗？ | 当客户优化成本时可能承压。 |
| 该公司是AI资本支出的瓶颈项吗？ | 当客户需要供应时可以重新定价。 |
| 该公司只是一个叙事故事吗？ | 在催化剂缺失期间易受打压。 |

成本项承压，瓶颈项重估，叙事型标的需要严格把握时机。

## 9. 非共识真相二：真实顶部早于指数高点出现

市场广度在指数见顶之前已经先行走弱。6月份的指数高点由Samsung Electronics、SK Hynix及少数AI基础设施股票驱动，而KOSDAQ和中位股票早已提前走软。

```text
中位股票走弱
+ KOSDAQ 走弱
+ KOSPI 存储超大盘股强劲
= 指数创新高与实际体验疲弱并存
```

下半年的关键观察指标包括：市场广度、涨跌比率（ADR）、KOSDAQ/KOSPI相对强弱、超大盘存储股交易额占比，以及KOSDAQ散户/机构联合买入情况。

## 10. 非共识真相三：估值倍数先于基本面充分兑现

Samsung、SK Hynix、Micron和Samsung Electro-Mechanics的基本面均表现强劲，但股票涨幅超出了近期盈利的增速。市场提前重估了估值倍数。

```text
旧有框架：
存储器 = 商品周期 = 盈利高峰折价

上半年框架：
存储器 = AI基础设施瓶颈 = 供应短缺叠加合同溢价
```

下半年的风险不只是盈利走差，还在于：若市场重新将存储器视为商品，即便盈利良好，估值倍数也可能下降。投资者应关注EPS预测上调速度、超预期幅度和2027财年能见度，而不仅仅是下一个季度的业绩超预期。

## 11. 非共识真相四：杠杆管道放大了尾部风险

韩国存储超大盘股不仅受基本面驱动，还受到杠杆管道的影响：在海外上市的2倍和3倍Samsung/SK Hynix ETF、国内杠杆与反向产品、期货及程序化资金流。

上涨正反馈循环（简化版）：

```text
股价上涨
→ 杠杆 ETF 再平衡买入
→ 期货与程序化资金支撑
→ 超大盘股进一步上涨
→ 更多动量资金涌入
```

下跌负反馈循环：

```text
股价下跌
→ 杠杆 ETF 再平衡卖出
→ 期货与程序化资金施压
→ 散户保证金与强制平仓风险
→ 更大幅度回撤
```

因此，即便基本面方向相近，韩国市场的波动幅度也可能远超Micron。

## 12. 非共识真相五：筹码从强手转向弱手

重要的问题不只是谁在卖，更是谁在买。

上半年，外资和部分机构账户削减了存储超大盘股的持仓，而散户和金融投资账户则承接了这些卖盘。

| 乐观解读 | 悲观解读 |
|---|---|
| 国内流动性强劲 | 筹码可能转移至较弱的杠杆持仓方 |
| 尽管外资卖出，KOSPI仍能维稳 | 若保证金平仓，强制抛售风险上升 |
| 大盘股买方基础扩大 | 也可能代表信息更充分账户的派发 |

下半年监控应综合观察外资现金流、散户保证金、存款、ETF持仓规模及程序化资金流。

## 14. 非共识真相七：营收模式比公司质量更重要

上半年，营收模式的质量比抽象的公司质量更重要。

营收模式较弱的标的类型：

| 类型 | 弱点 |
|---|---|
| 设备 | 订单空窗期可能导致盈利空窗期。 |
| 游戏 | 营收能见度在发布周期附近容易断层。 |
| 生物科技事件股 | 临床试验与授权空窗期造成叙事断层。 |
| 部分机器人与AI软件 | 预期走在营收前面。 |

营收模式较强的标的类型：

| 类型 | 优势 |
|---|---|
| 存储器 | 价格与出货量同步改善。 |
| eSSD与存储设备 | AI服务器存储瓶颈重估。 |
| MLCC与基板 | 服务器与封装瓶颈。 |
| 电力设备 | 数据中心电力需求叠加政策资本。 |
| 造船 | 长期在手订单能见度高。 |
| 国防 | 出口积压订单与政府预算。 |
| 大型生物科技平台 | 合同、产能与平台价值逻辑清晰。 |

下半年的规则很简单：

> 期权型标的缩小仓位，持续性营收瓶颈加大配置。

## 15. 延伸假设：韩国成为美国AI资本支出的杠杆代理

上半年，韩国市场的身份发生了转变。历史上，Samsung和SK Hynix是新兴市场、中国及全球IT库存周期的代理。而在2026年上半年，其与美国AI资本支出的关联性大幅增强。

盘中分析显示，美国前一日AI/半导体走势与韩国次日存储超大盘股之间的关联性更为紧密，中期相关系数估计为0.42。该数字需要正式重新验证，但价格行为与该方向一致。

问题在于分散化。若韩国表现为美国AI资本支出的杠杆代理，当全球AI因素降温时，国内缓冲作用将随之减弱。

另一个值得关注的议题是资本支出。盘中分析将SK Hynix的大规模资本支出承诺列为"峰值信心检验点"。具体金额和范围在成为核心论据之前应与公告文件进行核实。但从历史规律看，资本支出峰值公告往往在需求看起来最旺盛时出现。

下半年资本支出的关键问题：

| 问题 | 含义 |
|---|---|
| 客户合同是否为资本支出提供支撑？ | 降低供给过剩风险。 |
| 资本支出是否损害自由现金流？ | 盈利可能看起来良好，但现金流正在恶化。 |
| 供给是否在需求峰值附近到来？ | 引发2027年周期风险。 |
| 资本支出是否聚焦于HBM和AI存储？ | 必须与商品DRAM供给过剩风险区分开来。 |

## 18. 最终观点

市场可能会将2026年上半年铭记为韩国/半导体超级周期最佳年份，这并没有错。KOSPI翻番，存储器、基板、设备与电力板块的龙头股暴涨。

但更重要的故事在于：

```text
领涨范围极度收窄
估值倍数先于基本面充分兑现
外资卖出核心超大盘股，国内流动性承接
金融投资与ETF/程序化资金放大了市场波动
KOSDAQ缺的是买方，不是故事
美国AI领涨从GPU转向存储器、存储设备、服务器与电力
```

下半年的问题不只是"还有多少上涨空间"，更好的问题是：

1. 即便盈利强劲，股票弹性能否保持正向？
2. 国内流动性能否持续承接外资卖盘？
3. KOSDAQ买方能否回归？
4. AI瓶颈能否从叙事转化为真实营收与合同？

上半年是风险偏好上升期，但行情并不广泛——这是一个领涨高度集中的市场。下半年需要更高的精准度：真实瓶颈优于低估股票，持续性营收优于一次性催化剂，核心营业利润加自由现金流优于标题盈利数据。
