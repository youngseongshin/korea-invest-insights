## 개인정보 처리방침

최종 업데이트: 2026-05-15

Korea Invest Insights는 시장 해설, 리서치 대시보드, 교육용 투자 도구를 제공합니다. 이 문서는 `memory.koreainvestinsights.com`을 포함한 본 사이트와 관련 서브도메인 방문 시 어떤 정보가 처리될 수 있는지 설명합니다.

### 수집 또는 처리될 수 있는 정보

- **방문 정보**: 페이지 조회, 유입 경로, 브라우저 종류, 대략적인 지역, 접속 시각 등이 집계형 분석 도구를 통해 처리될 수 있습니다.
- **기술 로그**: GitHub Pages, Cloudflare 같은 호스팅·보안 제공자가 사이트 제공과 보안을 위해 IP 주소, user-agent, 요청 메타데이터를 처리할 수 있습니다.
- **댓글과 사용자 생성 콘텐츠**: 블로그 댓글은 Giscus/GitHub를 통해 처리될 수 있습니다. Memory Pulse 댓글은 닉네임, 댓글 본문, 작성 시각, 언어, 페이지 구분값, 스팸 방지를 위한 해시 처리된 IP/user-agent 값을 저장할 수 있습니다. 원 IP 주소를 공개하지 않습니다.
- **자발적으로 보낸 메시지**: 텔레그램, 서브스택, GitHub 등으로 연락하는 경우 사용자가 직접 보낸 정보가 처리될 수 있습니다.

### 쿠키, 분석, 광고

블로그는 가벼운 방문 분석 도구를 사용하고, Memory Pulse 대시보드는 Cloudflare Web Analytics와 공개 방문자 카운터를 사용합니다. 이는 사이트가 실제로 도움이 되는지, 어떤 페이지를 개선해야 하는지 파악하기 위한 목적입니다.

향후 Google AdSense 또는 다른 광고 파트너가 활성화되면 Google을 포함한 제3자 사업자가 쿠키를 사용하여 이 사이트 및 다른 사이트 방문 이력에 기반한 광고를 제공할 수 있습니다. 사용자는 Google 광고 설정에서 개인화 광고를 거부할 수 있습니다.

### 이용 목적

정보는 다음 목적에 사용됩니다.

- 사이트 운영과 보안,
- 집계형 방문 흐름 파악,
- 스팸과 악용 방지,
- 대시보드, 글, 탐색 구조 개선,
- 사용자가 자발적으로 보낸 문의나 정정 요청에 대한 응답.

### 제3자 제공

개인정보를 판매하지 않습니다. 다만 사이트 운영에 필요한 호스팅, 분석, 댓글, 뉴스레터, 스팸 방지 서비스 제공자가 일부 정보를 처리할 수 있습니다.

### 사용자 생성 콘텐츠

댓글과 공개 메시지의 책임은 작성자에게 있습니다. 스팸, 홍보방, 괴롭힘, 불법 콘텐츠, 투자 권유성 내용은 삭제하거나 숨길 수 있습니다.

### 문의

개인정보, 정정, 삭제 요청은 [연락처](/ko/page/contact/) 페이지의 채널을 이용해 주세요.
